演员徐贤通过MBC水木剧《时间》夯实了迷你系列剧主人公的基础。从一开始就出现很多闲话，一起合作的演员金正铉在制作发表会时拒绝和徐贤挽手、一直维持无表情的状态让他的态度争议扩散开来，之后金正铉又因为健康的理由，在电视剧结束前中途下车。徐贤需要填满所有男主人公突然消失留下的空白，主人公的分量无疑只能变得更大，承受那份分量并坚持到底成为了徐贤的课题。尽管多少是有些毛糙的申告式，但还算做得不错，在“徐贤的再发现”这样的评价中画下了句号。

采访翻译 | 徐贤2017年MBC电视剧《小偷家伙，小偷大人》终映采访
2017-11-23 18 min read # 电视剧 Drama # 徐贤 Seohyun # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

2007年，16岁的徐贤以少女时代成员的身份出道，之后通过发表《重逢的世界》《Gee》《Hoot》等热门歌曲在海内外受到欢迎。2013年开始，通过出演音乐剧和电视剧开始演技活动，而且最近在MBC周末剧《小偷大人小偷家伙》中饰演姜少珠第一次挑战地上波的女主角。虽然同时有着第一次担当主演的忧虑和期待，但徐贤自然的演技也得到了好评。尤其是和池贤宇搭档的爱情戏收获喜爱，得到了“姜石情侣”这样的爱称。电视剧结束后，与SM的专属合约也到期了，她独立了出来。
