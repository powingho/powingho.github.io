持续的崔秀英

以《Run On》《新年前夜》点亮年末和新年的崔秀英。
积累了许多经验后才成为的今天的崔秀英。

在一年即将结束的这个月里，你本人出演的电视剧开始播出了，电影也即将上映。电视剧《Run On》中饰演的“徐丹雅”是怎样的人物呢？

是大韩民国最厉害企业的继承者，但是因为是女性，她的继承顺序被推到了后面。丹雅对工作有很强烈的野心，工作能力出色，喜欢的和不喜欢的很分明，也明确地知道自己该走的道路。虽然因为说话直白经常听到别人说她很过分，但我认为丹雅是诚实的人，只是无法掩饰自己感情罢了。因为对时间敏感，所以在拐着弯子说话的时候只说有必要的话而已，省下绕弯的时间，希望更快地变得幸福。但是她遇到了打破自己严格遵守的秩序和规则的人，并爱上了他。

采访翻译 | 徐贤 2017年电视剧《小偷家伙，小偷大人》终映媒体联访
2017-11-23 18 min read # 电视剧 Drama # 徐贤 Seohyun # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

2007年，16岁的徐贤以少女时代成员的身份出道，之后通过发表《重逢的世界》《Gee》《Hoot》等热门歌曲在海内外受到欢迎。2013年开始，通过出演音乐剧和电视剧开始演技活动，而且最近在MBC周末剧《小偷大人小偷家伙》中饰演姜少珠第一次挑战地上波的女主角。虽然同时有着第一次担当主演的忧虑和期待，但徐贤自然的演技也得到了好评。尤其是和池贤宇搭档的爱情戏收获喜爱，得到了“姜石情侣”这样的爱称。电视剧结束后，与SM的专属合约也到期了，她独立了出来。
