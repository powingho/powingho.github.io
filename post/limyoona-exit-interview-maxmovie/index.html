原来寂寞的时候，所有人都一样

采访翻译 | 林润娥2019年电影《EXIT》宣传采访（maxmovie）

2019-07-27 6 min read # 极限逃生 EXIT # 允儿 Yoona # 少女时代 Girls' Generation SNSD # 采访 Interview # 电影 Movie # 翻译 Translation

穿着白色迷你连衣裙和运动鞋朝气地唱着《重逢的世界》，是对林润娥的第一印象。从那开始的12年后，少女时代的门面成为了国内发行公司期待作碰撞的夏季市场的主演。今年即将迎来30岁的林润娥，现在开始了她人生的第二幕。不要因为是熟悉的面孔就提前作出结论，即便出道已经超过10年，但她还有许多要展现的东西。

- 《共助》（2017）对演员林润娥来说是值得感谢的作品吧，打破了只想看起来漂亮的成见，《EXIT》会不会和《共助》一脉相承呢？意珠（林润娥）哭的时候，看到嘴角耷拉下来总是会笑出来。

觉得很有趣吧？真是万幸。好像成为我展现自己另一面的时机，少女时代活动的时候展现了许多明朗、漂亮的样子，而演技需要成为角色，而不是我，是展现和舞台不一样面貌的机会吧。《共助》的朴闵英和《EXIT》的意珠都是惹人喜爱、让我有共鸣的人物，这很吸引我。之后一看，却成为了喜剧的演技（笑）。

- 《EXIT》是龙南（曹政奭 饰）和意珠的灾难逃生记，为了活下去而奔跑的场面很多，听说了你想继续跑但因为身体跟不上而哭了出来的事，对李相根导演应该有话要说吧？

哈哈，其实李相根导演也对他说了“再来一次吧～”觉得很抱歉，用尽全力去跑的场面真的很累，之后到了根本无法行走的极限，喊Cut后我就直接瘫坐了下来，不过幸好现场有很多人在照顾演员的安全。

- 因为你跑得很快，曹政奭演员不是问了你“润娥啊，你100米跑多少秒”吗？好奇那个答案是什么？

上学的时候也有跑过，不过跑了多少秒倒是没有记录下来，我不是跑不了的类型，大概可以候补接力跑的程度？（笑）大概在中学的时候吧。我对短跑是蛮有自信的，长跑的话会跑着跑着就没有力气了。

- 曹政奭演员拍铁棒的戏是亲自上阵的，他对自己的肌肉看上去相当有自信。林润娥演员应该也有自己觉得自豪的场面吧？

我觉得是在讲过去龙南和意珠是登山社团成员的场面里出现的攀登的部分。离开拍还有2～3个月的时候，我和政奭哥一起接受了训练，当然也有用到替身，不过我和政奭哥亲自上阵的部分也很多，奔跑的部分都是亲自完成的。与其说是体力，不如说是以坚韧坚持了下来。不过因为灾难片是新的尝试，而且还可以展现利用身体的演技，所以我觉得蛮开心的。

- 意珠是展现年轻上班族苦涩一面的角色，从你小小年纪就开始社会生活的立场来看，应该会有很多觉得有共鸣的部分吧？

其实有很多人问了“没有经历过上班族的生活，能不能有共鸣呢？”这样的问题，但我们电影里有很多不是非得在特定的职场也可以感受到的经历和状况，我也一直有听一般人的朋友们说她们生活的故事，因此我自己的视野也拓宽了不少。托她们的福，演意珠的时候没有觉得有太大的困难。

- 不知不觉你已经满29岁了，少女时代出道就像在前几天一样，竟然就已经30岁了，时间真是过得很快啊。

嘿嘿嘿，其实我应该是满28岁。30岁临近前，烦恼也越来越多了。一直都过着忙碌的生活，第一次有了可以回顾自己的时间。以前是“Work-life balance”中的“Work”占了非常大的比重，那段时间可以集中在“Life”上吗（笑）？活动的时候，我虽然收获了很多，但可以完全审视自己样子的时间是不够的。去了和工作完全无关的旅行、也学习了语言和做了运动，因此现在适当地配合上“Work-life balance”了。

- 小小年纪就出道了，20代时基本上是没有私生活的吧。没有发生过什么事件、事故，可以称得上是少女时代的模范生吧。压力是如何舒解的呢？

我并不是那种模范生哦（笑），不过在很多人看来我有着正直的形象，因此在《共助》和《EXIT》中的角色看起来也觉得新鲜。舒解压力的方法没有什么特别不一样的，和朋友们见面、吃饭、聊天，挑一些和那天的心情或是天气符合的歌来听，这些对转换心情都挺不错的。唯一的乐趣是，我正在考虑有时间的时候去学个料理，虽然很感兴趣但还没试过。如果觉得非常烦恼的话，我会找身边的朋友聊天来舒解。

- 不久前上映的《女警》，同组合的成员崔秀英演员出演了嘛，好奇林润娥演员的观后感是什么？

我觉得和欧尼超合适的，（采访组笑了出来）不不不，不是说那些粗鲁的样子！是说欧尼把玫瑰这个角色演得很有魅力，哈哈！最近成员们都有很多个人活动，我们之间一直都是会互相理解和支持彼此的，是不是因为这样我们才能够长时间地维持很好的关系呢？啊，大家也都是会自己看情况好好去做的类型。

- 《EXIT》是CJ娱乐的夏季期待之作，预计会和宋康昊、刘海镇演员代表的其他发行公司的大片进行竞争，开心的同时也有很大压力吧？

哎一古，听了这话后感觉肩膀越来越放下了。首先，能够拍出第一部主演作品，我觉得很幸福，真的是和非常好的工作人员、演员们一起合作。抛开结果不说，我收获了很多，也积累了很多可以成长的经验。看到电影院里悬挂着我的大头海报，我现在还是觉得很神奇。（颁奖礼感言的口吻）非常非常感谢给我这样机会的发行公司CJ娱乐、制作公司外柔内刚、李相根导演、我的拍档曹政奭演员～哈哈哈。

（Sung Seon-hae 记者）
