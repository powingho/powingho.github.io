原来寂寞的时候，所有人都一样

2019-05-22 8 min read # 徐贤 Seohyun # 杂志 Magazine # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

无法预测

早早就开始漫长旅程的徐贤，不知不觉迎来了自己20代的最后一年，她怀着激动和兴奋的心情准备迎接连自己都无法预测的30代。

- 最近是怎么度过的？

正在特别的从容地度过中。以前总是把时间拆开来用，定好几点到几点要做什么事。从忙碌的行程回来的时候，如果没有属于我自己的预定计划的话，就会没有任何想法地活着，所以那时会把时间过得很紧凑。我现在让自己更自由一些，正在努力不受时间强迫地生活着。

- 那份努力有很好地实行吗？

好像是那样的。最近经常去运动，一周可能会去5次以上？每天早上会早早地去运动，运动的话确实感觉了活力，当然变得健康也是必然的。

- 总是过得很忙碌，突然多了很多时间的话，好像也不太适应，另一方面来说也可能会感到不安。

刚开始会是那样的，但最终那是我人生中必须要经历的时间。也曾经觉得焦虑，但过了一段时间后，心里变得轻松了不少。所以，挑选下一部作品也需要更长的时间，即使会晚一些，我也想以好的作品回到大家面前，因此希望粉丝们也不要感到着急。超过10年没有任何空白期地过来了，对粉丝们来说也是第一次经历这样的时间。

- 就像活动了很长时间那样，对粉丝们的心好像也会很特别。

和粉丝们的关系很久了，粉丝们也总是站我这边。活动的时候不可避免地会有觉得辛苦的时期，那时因为粉丝们才可以无恙地度过了。我枕头边堆了很多粉丝的信，每次觉得辛苦的时候都会看一看粉丝们送来的信。不是别的，即便只有一个人真心地喜欢我，也足以让我感到幸福，但却有着这么多人喜欢我，有时会觉得我会对他们的生活产生影响，因此感到责任很重，有时也会自己觉得是很有福气的人。粉丝们就是我努力生活的原动力吧。

- 爱豆的生活速度非常快，竞争也很激烈。和现在生活的温度差很大吧？

很大。回想起那时，会觉得真的是非常努力地生活着。好像因为那时的努力，才可以这样感谢地享受现在的从容。不久前我看了那段时间写的日记。很久以前开始，不管多忙的日子，我都会写日记，那时觉得如果简短的记录都没有的话，好像不会记得是怎样生活过来的了。看了以前的日记，发现都是写得满满当当的。哇，我那时真的是想不留遗憾地、努力地生活啊。

- 少女时代的忙内转眼就到了20代的最后一年了，从10代开始就在这个世界里。

我不觉得演艺界是和其他社会生活有很大不同的地方，都不过是人生。而且每年我觉得重要的价值观也会一点点发生变化，但除此之外，如果有绝对不会改变的价值观就是“最后的胜者是善良的人”，这也是我的座右铭。我觉得只要守住这一条，在这世上活着的时候就不会迷失自己。就算这个世界马上就毁灭了，不想后悔的话，就要过着对自己来说是堂堂正正的人生。

- 但是要善良地活在这世上其实并没有那么容易吧，人际关系也是随着对方的不同，只能做不同的反应吧。

是困难，非常困难。所以，我才有了这个座右铭，如果没有这样坚定的座右铭的话，似乎会被生活中遇到各种变数所动摇。任何人不是随时都会死去吗，到那时比起给子女、父母、朋友留下我是什么样的人的印象，能够跟自己说我是过着这样的人生更为重要。出道后不久，我就自己定下了座右铭，虽然是经历很长时间的练习生生活才出道，但小小年纪就开始社会生活是不容易的，还没成熟的我对所经历的事感到很吃力，我苦恼着要怎样去克服。那时便开始看很多书，书里的人物忍受辛苦的事活着的样子成为了我的力量。

- 今年对于徐贤来说是20代的最后一年，也可能是为了人生第二幕而调整呼吸的时期吗？

我觉得现在好像也是开启人生第二幕的时期，但是与其想象会发生什么了不起的事，不如像到目前为止所做的那样，不忘初心。我也不会着急。目前为止是，努力去做眼下要完成的事，抱着把事情做好的心态跑完短跑，现在为了长距离的奔跑，休息、加速、又重新调整呼吸，这样生活着。我想快点到30岁，茫然地期待着30岁会不会变得更从容、幸福呢，也想知道会发生怎样的事情。

- 最近好像活动的领域又扩大了，不久前在颁奖礼上担当了司仪，也进行了特别专辑的工作，还有之前完成了电视剧。在不同的领域里活动，有没有觉得自己的能力范围扩大了呢？

不是只会唱歌的徐贤或只会演戏的徐贤，有机会向大家展现不同的样子是值得感谢的事情。发现新的我对我自己来说也是很开心的事，另外，因为新的活动而遇到的人们也觉得很珍贵。进行不同的活动虽然从职业角度来说很不错，但从平常人角度来说也是很好的事。

- 每个人活着的时候，都会经历极度辛苦的事情。遇到意想不到的难关时，徐贤会如何对应？

我很坚定地相信自己。如果遇到那样的瞬间，责任感会变得更坚定。带着不能倒下的想法，积极地去为了寻找一个支点而努力。我相信如果克服了这一瞬间的话，会变得更加坚强。

- 有什么是让5月的徐贤觉得心动的吗？

新芽！新芽长出来的时候，或是花蕾盛开的时候，我都一定要拍下照片。看着那些，我感觉很幸福。绿油油的颜色很好看，也感觉到了生命力。我喜欢现在的季节。

- 徐贤的季节现在是在哪呢？

是春天，新的春天。季节不是会一直轮转的嘛，新的春天正在开始。

- 春天的徐贤想尝试做什么样的事情呢？

想集中在演技上。目前为止还没有拍过电影，话剧也希望尝试看看。

- 虽然总是舞台上的主人公，但在话剧或是电影里可能不是主人公。

我不是想做主人公，只是希望可以有趣地去演戏。我希望可以尝试客串出演，也愿意试试小的角色，如果有机会的话，不管是类型还是角色都没有限制地想去试试看。事实上，大众所认识的我不是特别有限的吗，在少女时代里是忙内，作为忙内能够扮演的角色也是有限的。出演的综艺节目并不多，所以能够展现我的机会很少。以后希望可以展现我不同的面貌，作为演员挑战多种多样的事。

- 20代现在还剩下半年，想如何填满剩下的半年时间呢？

会很有意思的。不知道会去往哪里，谁都无法预测的那种感觉。在意想不到的瞬间，在大众面前突然登场，然后开心地度过。
