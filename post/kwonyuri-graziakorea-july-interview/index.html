原来寂寞的时候，所有人都一样

首页 归档 标签 关于

采访翻译 | 权俞利2018年杂志《Grazia Korea》7月号

2018-06-19 9 min read # 俞利 Yuri # 杂志 Magazine # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

Q 画报好像很久违了，这段时间怎么度过的？

Yuri：拍摄了预计下半年播出的电视剧《心里的声音2&3》，大家都知道《心里的声音》已经连载了10年的网漫，而且得到很多喜爱，所以我觉得蛮有负担的。但是作为很有趣地看了原作的粉丝，这也是让我有了野心的作品。我能够来表现主人公爱凤开朗的活力这本身就是感到光荣的事情。

Q 变身成爱凤好像并不容易？

Yuri：虽然很开心，但也感到负担。因为拍摄现场的气氛就像原作一样轻松愉快，与其说是表演，不如说是气氛正好让我可以很自然地融入了爱凤这个角色。对我来说是有很多愉快记忆的作品，因为是无论何时都想尝试的作品类型，很幸福地拍摄来着，现场那样的气氛如果可以很好地通过作品传递出来就好了。

Q 《心里的声音2&3》看点会是什么呢？

Yuri：在这次作品里想做到“这样毁掉自己的形象也可以吗？”“这样放下自己也可以吗？”这种程度，没有顾忌地去表演了，所以我自己根本不敢看监视器（笑），请期待爱凤让人觉得甜蜜又让人觉得吓人的表现吧，尤其是和平时的我完全不同，希望可以愉快的收看。

Q 作为歌手就已经出道的第12年了哦？

Yuri：去年出了10周年专辑之后又1年过去了，出道那时我真的想不到可以走到现在，某个瞬间成为得到后辈们的问候的前辈，现在依然觉得很神奇。10年间一直在活动，感到辛苦的时候虽然也很多，回过头看看，发现是得益于我们的成员才可以走到现在，彼此互相补足了很多，因为个人活动繁忙，我觉得辛苦的时候，其他成员补上了空缺的位置这样我们才能走到现在。

Q 这样看的话，出道那时是分配以舞蹈为主对吧？

Yuri：是的（笑），和允浩偶吧是参加的同一个选拔被选上的同期。以舞蹈为主的成员实际上没有变过，但大家比较熟悉少女时代全体的刀舞，所以我的分配好像不太明显了。活动越久，谁分配什么就变得没什么意义了。C位啊门面啊都是轮流来。

Q 当时是男艺人代表性的理想型吧？

Yuri：《说出你的愿望》的那个时期，那时是我站C位。当时突然在节目里或是采访里听说把我当作理想型的艺人好像蛮多的，但那时的我太忙了，所以都不怎么知道。觉得很神奇，也很感谢。

Q 一直在活动这件事本身就挺了不起的

Yuri：直到现在还是感到辛苦的话，精神上会有动摇。对我的未来有很多想法，苦恼着怎么做才好，变得更小心了。现在也不是盲目地挑战新事物的年纪了，所以也是抓住重心有点困难的时期。

Q 平时看起来觉得是娱乐行业最适合的性格

Yuri：并不是哦，我反而是会容易受伤的类型，经常觉得这份职业会很吃力。不管任何行动的反馈本来就很强烈，要最快速度地接受，这件事做起来并不容易，因为是称赞或是非难都会得到很多的职业，要淡然地接受这一切需要付出很多努力。

Q 举例来说，需要哪些努力呢？

Yuri：工作本身就不是固定的，如果心态控制不好的话，身心都会垮掉。所以需要去努力坚持自己定好的日常生活模式，这样才能坚守自己的价值观，平时最大限度地安排日常的行程，然后跟着定好的日程去完成。不管是发生特别好的事情或是感到特别辛苦的事情，都会维持好中心，不会被影响太多。

Q 回头看的话，觉得最幸福的瞬间是什么时候？

Yuri：用手指数出来的话还蛮困难的，获得大赏的时候，面对5万名粉丝演唱的时候，作为代表亚洲的女团站在舞台上的时候。作为少女时代享受无上荣光和荣誉的时候，反而好像要为了变得淡然接受而努力。“在这世上感到最幸福的时候”这样的概念本身是没有的，但少女时代出道的时候是我一生中印象最深刻的瞬间。所以，觉得每一天都做得很好地度过应该更重要。

Q 与一般的同龄人对比，不是度过了华丽的时期吗？

Yuri：每当那个时候都会努力简单地生活，如果考虑很多变得太真挚的话，应该会很累，所以我没有过“如果我不是这种方式活着会怎样呢？”这样的想法，就那样接受我所处的现实。因此，我很喜欢采访，会有一些平时不曾有过的想法，而且可以记录下来，之后翻看我所做的采访报道来看，会觉得“我还说过这样的话？”。

Q 有属于Yuri的座右铭吗？

Yuri：不至于说是座右铭，是我喜欢的一句音乐用语，意大利语是“A piacere, ma non troppo”，自由但不过度放纵的意思。

Q 最近休息着有什么想法吗？

Yuri：可能身体一直处于忙碌的状态，所以即便是有了休假的时间，也不能完全停下来，身体感到不适才能强制休息吧。不久前我自己觉得需要充电了，所以强制自己休息下来，但是该从哪里开始、怎么做、要做什么这些选择起来也是不容易的。下定决心“好好做吧”，但并不奏效，只有在大众面前出现的我和我得到喜爱的时候，感觉我才是存在的，产生这样的错觉，有点遗失自己了。

Q 出道已经超过10年了，好像苦恼变得更多的样子？

Yuri：但是，仔细想想，不管什么状况都会想到“我”是不是存在的，不是嘛？为什么呢？我看了从练习生时期开始写的日记，不管是17年前，还是10年前，苦恼的事情都是一样的。我一边反省着依然没有变化的样子，一边想着人生大概就是这样的吧。不要那样苦恼，尽情享受比较好，所以得出的结论就是完完全全地去感受每一天地过着就可以了。喜欢我的人们、我喜欢的人们，和他们一层一层堆叠一起的时间是最为重要的。

Q 乐观的性格有所帮助吧？

Yuri：让我受伤的人或是让我感到受伤的瞬间，我是会很快忘掉这些的类型，这好像是与生俱来的天性，我觉得这样的性格在这样的工作当中是相当大的优势，但另一方面是无法掩饰自己的感情好像就变成了劣势，看到什么就是什么，在节目中可能会成为劣势吧，觉得无趣的话就会写在脸上了。

Q 回首过去10多年的生活，是什么感觉呢？

Yuri：了解我的人、理解我的人即便只有一个也会感到幸福，从这一层面来说，能够在SM里、和少女时代一起度过对我来说是最好的祝福。依然相信我的可能性给予支援，不管是公司、经纪人还是成员们，从这一层面我觉得很满足。虽然是常常见到的人们，好奇彼此的感情、关心彼此的这种关系我很喜欢。

Q 做着访问，好像看到了Yuri新的模样

Yuri：做的访问多了，我想应该会有更多机会面对一个或许不太熟悉的、或许对大众来说比较不一样的我。感觉采访越做我也越能了解自己，随着层层堆叠，我可以寻找和变成更权俞利的样子。这么说的话，做着访问的时候我是不是太真挚了？（笑）

Q 对于剩下的2018年，可以告诉一下有怎样的心态或是计划吗？

Yuri：2018年已经过了一半了，太快了！2018年上半年因为爱凤的角色需要，体重增加了5～6kg，回到首尔之后要尽快开始减重了，找回健康之后，会投入更多时间通过优秀的作品和大家见面，那些没有尝试过的事情也希望可以挑战一下，各个方面都希望更多地学习一下，料理啊插花啊，另外在书店里到处转悠的Yuri也可能会遇到。
