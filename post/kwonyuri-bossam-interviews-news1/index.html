采访翻译｜权俞利2021年MBN电视剧《打包袱-盗取命运》终映采访（news1）

2021-07-07 6 min read # 绑匪偷走命运 Bossam Stealing Fate # 俞利 Yuri # 电视剧 Drama # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

成功结束《打包袱》的权俞利最近进行的网络采访，表达了送走水镜的感想。这是她的第一部古装剧，饰演感情幅度变化很大的水镜对权俞利来说并不是轻易的过程，说着希望像有主体能动性的水镜一样，流露了对作品和角色的深深喜爱。

“虽然不管哪部作品我都很喜欢，但这次遇到水镜，真的是哭着笑着感受到和学习到了很多东西，很期待去拍摄现场，因为是非常期待的作品，这么快结束了以至于我仍然没有实感。”

《打包袱》创下了MBN电视剧史上的最高收视率记录，获得一些不错的成果。

“我们是以事前制作的形式来进行拍摄的，虽然感受到了和对手演员的合作会有很好的协同效果，但结果（收视率）又是另一个领域的事了，不是吗？虽然希望可以将好的协同效果顺利地传达给收看的观众们，但得到的称赞比我想象的还要多，我觉得非常的欣慰，这些反馈对于一起工作的同事们也是一种报答，我很感谢。”

权俞利表达了对水镜这个人物的深厚感情，虽然进行了很长时间的演技活动，但是她知道遇到这样主体化、能动性的角色并不容易。

“收到剧本的时候，我就被水镜迷住了，太喜欢这个角色了。是一个有能动性、主体性的翁主，还有着温暖的品性，是我很想成为她这样的女人。能有机会饰演这样的角色，我很开心，很高兴遇到让我心脏扑通扑通跳的人物。水镜是会说着‘只有我死了，所有人才会安全’的翁主，我有想过如果是一般人的角色，会做到这样吗？这样坚韧的人物真的是非常特别的角色，有着这种感情变化幅度的角色，真的不是轻易有遇到的机会，所以我更被吸引了，说了‘我想演’之后，在回家的路上顿时也感到了害怕，只是因为我想演这个角色，但不确定自己能不能演到让别人不会想起别的演员的程度，而且我并没有出演古装剧的经验，所以也非常担心。直到开始第一次拍摄的时候，这种‘担心’都还是占据着主导地位的，不管是导演，还是制作组的同事们，我甚至去找申东美姐姐问她我该怎么办，因为不知道该怎么办，我还去做了苦恼商谈，还哭了很多次。”

回想起第一次去拍摄现场的那天，权俞利又一次深深地陷入那天的感情之中。

“因为我非常地不安，看着我看剧本和准备拍摄的样子，大家似乎也觉得我看起来特别地不安。导演说（对于水镜）希望我像一个遮着一层面纱的女人去表演，但是我不太清楚那是什么样的感觉，所以初期不太表露感情的样子演起来特别困难。”

准备过程也不简单，她经历了四次试镜和无数次的剧本练习。

“导演说和我第一次开会的时候，从对话中觉得我的活力很好，说那种活力和水镜身上的活力很像。我也希望能和水镜这样的人物一样，那个角色所拥有的坚韧的活力和领袖气质，是我想要学习的女性形象。导演在非常小的地方发现了相似的样子，也似乎相信我的潜力，我真的觉得非常地感谢。”

在成功结束第一次古装剧的现在，作为演员有没有获得自信？

“我每一次的作品都不一样，角色也不一样，所以可以轻松升级的话就好了，但是在这次作品中，我至少可以挑战不同的题材，是这样的心情。“

“没想到权俞利演得这么好“这次作品当中尤其获得了很多关于演技的好评。

“这让我吓了一跳，通过各种作品慢慢地积累经验，（获得好评）让我觉得非常地感谢。虽然我努力地不表现出来，但有时我会比别人更不信任自己，自信心也下降很多。庆幸的是发现原来不管在哪都有人一直关注着我、支持着我，这给了我力量，对我的信任也似乎一点一点地产生了，所以我觉得很欣慰和有意义。现在还有很多需要翻越的山，也有很多想要尝试的事情，尽管缓慢，我也会慢慢地去实现。”

《打包袱》之后有什么计划呢？

“让我感到庆幸和感谢的是，现在收到了出演作品的邀请并正在讨论当中，虽然目前没有任何决定，但我在寻找能够在不久的将来尽快和大家见面的方法。想尝试爱情喜剧，更有生活感的角色和让心脏扑通扑通的台词，我想挑战有着这样内容的爱情喜剧。”

一直都有传言少女时代要重聚回归。

“因为少女时代是像故乡一样的地方，我觉得是不是总有一天会回去的呢？我们成员们在各自的领域长期地、尤其是最近特别旺盛地进行活动，各自都有在这段时间积累的经验，当我们再聚到一起的时候，不管是音乐或是站上舞台肯定会变得更有意思。为了能够呈现精彩的舞台，我们和往常一样认真地在讨论关于回归的事。少女时代应该也会有好消息的吧，我也在期待着。”
