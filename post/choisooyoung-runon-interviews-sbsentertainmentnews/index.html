采访翻译｜崔秀英2021年JTBC电视剧《Run On》终映采访（SBS演艺新闻）

2021-03-14 13 min read # 奔向爱情 Run On # 电视剧 Drama # 秀英 Sooyoung # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

偶像出身的演员为了得到演技的认可，比想象更激烈地苦恼和努力。从练习生时期开始接受系统性的表演课程，偶像们自己对演技也是充满热情，对待演技的态度非常认真。以前偶像以自身的认知度更容易获得机会，以无法容忍的“蹩脚演技”让人皱眉的情况发生过很多次，但现在不一样了，他们积累了很多努力，现在戴着有色眼镜看待偶像出身演员的情况确实减少了。

崔秀英在偶像的演技被严厉评价的那个时期开始了演戏，不管是谁都因为“演技争议”受苦的时候，崔秀英离那样的争议很遥远。本以为她会融入剧情，展开自然的演技，虽然不是很突出，但是角色的消化能力非常稳定。 从练习生时期就开始准备演戏的崔秀英在少女时代出道的同时，作为演员不断敲开作品的大门，不知不觉间演员经历已经超过了10年。

通过电影《纯情漫画》（2008）、《Girl Cops》（2019），电视剧《第3病院》（2012）、《恋爱操作团》（2013）、《38师机动队》（2016）、《如实陈述》（2020）等作品，崔秀英作为演员踏出了有意义的步伐。虽然得到了演技无可挑剔的评价，但感觉还是差了2%，因为一提到“演员崔秀英”没有能立即想起来的强有力的一击。即使不是热门作品，也会出现一个充满魅力的“人生角色”，那个缺失总是让人遗憾。

但是，现在可以抛开这种遗憾了，因为最近剧终的JTBC电视剧《Run On》的徐丹雅。 徐丹雅虽然稍微扭转了过往电视剧中的财阀女角色，但她拥有独立、活泼可爱的魅力，崔秀英用自己的特色完美地演绎了出来。徐丹雅越有魅力，越有人称赞她“崔秀英的演技是这么好的吗？”崔秀英的“人生角色”终于诞生了。

崔秀英对《Run On》和徐丹雅的喜爱，很大原因是她自己也获得了“安慰”。因不理解自己努力的人而受到伤害并产生怀疑，这部作品让她明白了“信任”的价值。

《Run On》中的奇善谦（任时完 饰）握着因伤放弃田径的金宇植（李正河 饰）的手说：“我来试试看，只要你肯相信我，我就帮你做到。” 崔秀英在这句台词中感触颇深，相信自己努力的他们，相反的，当自己相信他们并一起工作时，感受到了会带来多么幸福的结果，这才得到了安慰。

对她来说最需要的是，相信并关注“演员崔秀英”。给予信任后，她回答了像徐丹雅一样充满魅力的角色。这些小小的信任汇聚、积累，崔秀英是否也会更接近“信且看的演员”呢？

-《Run On》中的徐丹雅这一角色和以往电视剧中出现的女性角色稍微有些不一样，好奇你在刚开始饰演徐丹雅的时候是什么感觉？

我是很胆小的人，饰演丹雅的时候有让我觉得害怕的地方。丹雅这个人物如果一开始演不好的话，看起来就会很无聊，这让我害怕过。丹雅是一个让人无法讨厌、有钱但却有正义感、理念和思想都觉醒的朋友。因为她是最近时代可以看到的新女性角色，所以有了让人觉得崭新的地方。我相信编剧可以从始至终不让角色变质地解开丹雅的成长故事。所以，虽然我一开始觉得害怕，但渐渐地变得可以无所畏惧地去表演了。

-对于那样胆小的崔秀英来说，从什么时候开始可以自在地靠近徐丹雅？

我从一开始就对丹雅觉得蛮自在。丹雅特别的语气会让人觉得不自在，观众们会怎么看待呢、会不会觉得碍眼呢，我一直都在怀疑这些。但是丹雅那么说话的原因，不是为了伤害对方，而是她本身就是这么想的才这样说话，这么想之后我就觉得自在了。丹雅的语气中分明有着编剧的用意，我这样相信着去表演，也变得舒服多了。

-徐丹雅和李映禾（姜泰伍 饰）的爱情线，颠倒了一般爱情线中财阀男主角、糖果女主角的俗套设定，带来了很多趣味。从演戏的立场来看，好像会感受到与众不同的有趣？

从演戏的立场来看，我倒没有觉得改变了原来俗套，就只是觉得因为是徐丹雅才会有的台词而已，播出之后有很多人说好像男女调换了，但比起那些，丹雅面对强者时无所顾忌、无限强硬的模样，让我有新的感触。对我来说，丹雅是一个特别有魅力的朋友，表演的时候觉得特别有趣。不管怎样，相对于奇善谦（任时完 饰）和吴薇朱（申世景 饰），呈现丹雅故事线的时间是有限的，这让我产生了希望观众们能等到丹雅出现的野心。所以每次出现的时候，我都想搞笑一些、造一些值得等待的亮点，想要去呈现那些，看来在表演的我也确实觉得很有趣。

-徐丹雅和现实中的崔秀英有多像？还有好奇崔秀英理解的徐丹雅是个什么样的人物？

喜欢工作、追求完美这一点来说大概有70%的相似，但我并不是像丹雅那样不知道会做出什么的性格。我是不是比丹雅更会看人眼色、更会照顾别人一些呢？（笑）丹雅不是面对强者时变强、面对弱者时变弱的人，只要是不符合她所认为的正义和标准，她就会变强硬，是个对自己的标准很明确的人。所以她认为自己树立的标准就是正确答案，而实践那个正确答案的时候总是被误会。大家都害怕和丹雅对话，但丹雅成为真正的朋友的话，是会比任何人都更照顾自己的人，比任何人都更认真对待工作的人。

-和一起出演的任时完、申世景、姜泰伍合作配合得很好，是因为年龄层相似所以相通的地方比较多吗？

即便是同龄演员，但因为彼此性格不同，也有一起演戏时会犹豫的人。相反的，尽管是前辈，也会营造无拘无束、自由自在的让人舒服的氛围。做着这份工作的时候，我遇到了很多不同类型的人，不会因为是和同龄演员一起演戏就觉得“太好了”。但是，这次电视剧集合了性格品行特别好的演员，所以真的很好。在现场我们真是无拘无束地交换对于拍摄场面的想法，分享彼此的构思。虽然是同龄人，但我和看着前辈们时一样，和这些演员们相处的过程中，我学到了很多东西，真的是遇到了充满尊敬的演员们，然后一起工作。

-和申世景演员也是大学同期，好奇和申世景在现场的氛围是怎样的？

因为世景是大学同期，我们真的认识蛮久了。当然因为各自繁忙的工作，过去没能经常一起聊天，但为什么会有那样的关系呢？只是在远处看着，也想为那个朋友所走的路应援，也好像知道她是以怎样的心情去面对的，这样的关系。在远处看着世景开拓自己道路的样子，让人尊敬。果不其然，一起合作后发现她就和我想的一样，是一个品性、实力都特别特别好的演员。我向世景学到了很多，幽默感也很合拍，所以丹雅和薇朱的场面表现得很好。

-徐丹雅是一个坚守自己所划定的界线和标准的人，崔秀英也会定下“要是这样的话不行”这样的界线吗？

我是一个对自己严格的人。不管怎么说，因为过着暴露在大众面前的生活，所以即便是展现很少的内容，我也会在要呈现的东西上花费很多心思，还会制定作战计划，所以我应该是有着“没有准备的东西，不管是什么都是无法成事的”这样的界线。还有，我觉得自己是一个懒惰的人，定下了那样的界线，才是让自己（不轻易倒下）的基本装置。

-饰演徐丹雅的时候，有没有其他人帮助你分析角色或者给予表演建议的？

对于角色分析和表演建议，我想导演和编剧应该是帮助我最多的，也是最准确的。在写丹雅英文台词的时候，得到了Tiffany的帮助，比起直接用英语来翻译，我想呈现丹雅特有的才气和角色个性。Tiffany本来就是当地人，也是学了很久表演的朋友，我跟她说明了丹雅的剧本和前后状况，她就流畅地写出了英文台词。让丹雅英文戏的品味得以完成的是Tiffany，我一定要跟她说谢谢。

-《Run On》描绘了2030年龄层的年轻人们成长，你应该也回想起崔秀英的青春吧？

和Tiffany聊天的时候，我们说过“丹雅好像少女时代”、“是不是很像我们？”为了守护属于自己的东西，有点像是站在刀刃上的样子，和我们很像。《Run On》中各个地方都隐藏着想对年轻人们传达的话，我也是在看剧本的时候，获得了很多安慰。特别是“我来试试看，你肯相信的话，我来帮你做到”这句台词，这部电视剧（对我来说）就像那句台词一样的存在。“演员这样相信我，那我就得好好地让它完成才行”编剧这样的心意，似乎通过那句台词来传达了，我觉得非常感谢。我也是一直抱着“相信我的话，我就能做到”这种想法活着的人，这句台词好像很好地表达了我的座右铭，所以我看那场戏的时候哭得很厉害。

-开始演戏后，一直以来慢慢的积累作品履历到现在已经超过10年了，对于演员崔秀英，《Run On》是留下怎样意义的作品？

《Run On》是让人类崔秀英获得更多安慰的作品。少女时代活动的时候获得了很多喜爱，对于得到的大众反馈，我并不是会悲喜交加的性格。但是，做着这份工作的过程中，我会有点怀疑，觉得人们似乎不能正确对待人类崔秀英的努力，不是对大众，而是工作时遇到的人让我感到怀疑。把演员的努力当成是理所应当的去对待，有时则是把我的话往与我意图不同的方向错误地解释，我也看到过很多误传。为了守护我自己，我把自己的心门关上了，相反地也假装打开过。这部作品似乎给我传递了“你肯相信我的话，我会试试看，成为帮你做到的人”这样的讯息，这是一部完全地相信我并交给我，而我信任的时候，让我看到出来什么结果的作品。所以，以后遇到其他作品似乎也可以因为回想起这部作品得到的感动和想法而去投入。

-丹雅虽然在工作上很完美，但在爱情上却像孩子一样笨拙，现实中崔秀英的爱情好像是不一样的。如果崔秀英给丹雅爱情建议的话，想告诉她什么话呢？

那个人的生疏感当遇到能够抚慰的人时，爱情似乎就完成了。并不是因为自己做得好才能恋爱很久，而是遇到可以彼此照顾的对象时，恋爱才能维持得很好。映禾正是因为丹雅现在的魅力才喜欢她的不是吗？好像不需要别的建议，只是丹雅不知道怎么接受罢了。所以，映禾为丹雅弄的生日派对，才会让她觉得非常不自在。这是和我有点相似的一面，因为我也是独立的性格，所以别人说要做什么的时候我就让他不要做，收到多少我就会还多少回去。最近则是反复告诉自己对获得的东西宽容一点吧，所以我想和丹雅说“享受映禾带来的喜悦吧，也请他吃饭吧，生日派对也开起来吧”。

-徐丹雅是一个特别有魅力的角色，也获得了“人生角色”的称赞，所以更好奇崔秀英的下一个角色了。有什么角色是你想挑战的吗？

我最近才发现自己不是一个开朗的人，所以想试一下阴森一点的的角色，或是稍微神秘一点、黑暗一点的人物。另一方面，因为我很喜欢把拍摄场面弄成搞笑的，所以也想挑战一下真正的喜剧，但是我也喜欢像《Run On》这样审视生活，有着现实讯息、安慰讯息的作品。比起定好要演什么样的角色，我更希望成为对给予我的事全力以赴的演员。
