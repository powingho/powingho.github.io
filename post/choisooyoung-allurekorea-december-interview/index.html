2020-11-20 8 min read # 新年前夜 New Year Blues # 奔向爱情 Run On # 电视剧 Drama # 杂志 Magazine # 秀英 Sooyoung # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

为了深深地扎根，秀英

虽然比任何人都更严格地对待自己，但也拥有一颗懂得接纳的宽容之心。今天也是不断前进的演员崔秀英。

-天气突然变得寒冷了，你喜欢冬天吗？

虽然我的生日在冬天，粉丝们也会叫我冬天的孩子，但实际上我很怕冷，庆幸的是我并没有很多在冬天拍摄的作品。

-电视剧《如实陈述》是例外吧？

是的，那时是我第一次经历冬天的拍摄现场，但运气很好，去年冬天并不是很冷，动作戏或奔跑的场面很多，没有感到寒冷地度过了。

-在12月将要播出的JTBC电视剧《Run On》中饰演“徐丹雅”，是什么样的人物呢？

徐丹雅是大韩民国最厉害企业的唯一嫡系，但因为是女儿而被排在继承顺序靠后的位置，所以她一直生活在竞争的构图里，总是很紧张，性格非常直白，在某些方面来看就是不把别人放在眼里。对于自己过分的行为，她觉得对大家都是有帮助的，因此更被人讨厌了，她遇到了对这样的她不以为然的男人，让她一直坚持的紧张和戒备被打破的人。

-不管扮演的角色是什么，你都可以爱上那个角色吗？角色和演员的关系对演技也有影响吗？

是的，对角色完全的理解，还有在扮演那个角色的时候，如果没有制造动机和契机，会很难演好。据说很多饰演反派角色的演员，在分析角色的时候，会从积累对角色的怜悯开始。

-编剧创造的剧本里的角色通过演员重新诞生了，崔秀英表现的徐丹雅会是什么样的呢？

刚开始在剧本里看到的徐丹雅，是比现在更冷漠的一个人。和崔秀英相遇后丹雅变化的地方，是感情的起伏更大了。相比编剧的意图，我觉得自己更利他地去表现了。

-《Run On》是相爱的人们以各自的语言进行沟通和建立关系，你觉得自己是擅于沟通的人吗？

我觉得自己现在是擅于沟通的，在变成这样之前经历了相当艰苦的一段时间。刚出道的时候完全就像小孩子一样，小孩子对“热锅是危险的”没有概念，一直会去触摸、受伤、动摇，对吧？但另一方面，我也担心这样的行为会被别人看到，所以也曾经藏得严严实实的。

-开始拍摄作品的时候，有过什么特别的觉悟吗？

我在作品结束后一直都会想象自己和角色的相遇，“和角色相遇的时候会觉得害羞得逃跑吗？”“会听到对角色的感谢吗？”等等。把角色想象成实际存在的人，压力会特别大，但我会利用那些压力。

-是说压力会对表演好有帮助吗？

没错，现在回想的话，我过去饰演的角色中，有尽全力却没能表现的角色、有觉得抱歉的角色，相反也有着笑着问候送走的角色，有想偶尔见面、愉快地聊天这样的朋友。对待角色的时候，如果产生这样的心理压力，对爱上角色是有帮助的。

-本人好像是会对自己特别紧逼的类型？

像是折磨对吧？对的，我好像是喜欢折磨自己。

-作品结束后，从角色中走出来也需要时间吧。

我觉得那个过程也很爽，（笑）那种孤独，甚至是犹豫，所有的感受。

-很帅气呢，好像也有消散负面情绪的方法。

很多时候我都是放任不管，有时会一直沉迷其中地生活着、有时也会偶尔忘记、有时会留给时间去解决，有时也会因为突然有了别的事情而彻底消散了。幸运的是，并没有很多让我长时间困扰的角色，好像都在适当的时候很好地走出来了。

-不管是谁都会有陷入低谷的时候，你有那样的经历吗？

在这个领域里，让人兴奋的事和没有让人兴奋的事是明确存在的，如果一直遇到不能兴奋起来的事就会变成低谷吧，为了我在工作的人、志同道合的人，让他们的活力进入我的体内，转换成那样的想法和感情，会有一种重新开启的感觉。表演的时候也是差不多的，即使做足了准备，也有不满意的时候，那时最快、最有效果的就是接收对手演员的活力。从身边的人获得活力、担起责任感，然后回报更好的模样。

-结束作品后，有为了回归日常而做的事吗？像是某种意识那样的？

去旅行，或是拍摄期间不能做的染发、美甲这些，和大家都是差不多的。

-回到日常的普通人崔秀英，和舞台上的崔秀英是不一样的吗？

相比站在舞台上的时候，我平时生活的状态会更紧张一些。我是比其他人会更严格地要求自己，但在舞台上我是不顾后果的类型，因为是以角色去面对的，进入某个故事里，只要集中精力去表现就可以了，所以我觉得舞台很棒、很舒服。

-因为是名人，回归日常也有很多的限制。因为会被人认出来而不能去做的事中，有觉得可惜的吗？

只想到一个，就是面对面交易！

-《Daangn Market》那种吗？

对，就是那个！收拾行李的时候，有些东西舍不得扔掉，一直想着要给谁，但又觉得别人不满意的话会成为负担，所以很犹豫。因为新冠疫情的关系，没有办法搞跳蚤市场那些。

-虽然无法享受面对面交易的乐趣，但本人的日常生活也是觉得享受的吗？天气好的时候，突然去兜风，或是去有阳台的咖啡厅喝咖啡。

当然是的，我喜欢制造氛围的事情，而且不会犹豫。下雨的时候会去吃葱饼，做美味料理的时候会叫上朋友们一起喝上一杯红酒，所以即使是生活在紧张之中，也还是健康的。

-看起来是很会照顾身边朋友的性格。

是的，但有一天我的朋友Tiffany告诉我要为自己而不是别人生活，为了我自己挑选满意的衣服来穿、为了我自己点燃蜡烛、为了我自己用漂亮的碗碟盛装食物，不要为了照顾别人、为了给谁看到，而是为了我自己去计划事情，而且她后来还买了漂亮的睡衣、内衣和耳饰作为礼物送给我，她明知道那些是我只会想想而不会去买的，不管怎样为了让我试一试才买给我的吧。（笑）

-那样做了之后有变得不一样吗？

变得更爱我自己了，我自己绝对想不到的东西，朋友会告诉我，所以才更喜欢。

-到现在为止，这个真的做得很好。如果想这样自己安慰的事，会是什么呢？

不想笑的时候，打起精神再笑出来。散漫地放松自己后，能在关键的时刻瞬间回到原来的位置。还有更重要的是耐心，耐心是团体生活给我的最大财富，也是我的武器。

-想起崔秀英时，想到的不管是什么都做得很好的八面玲珑的形象。

我自己也是那样想的，“就算我没有一样是特别突出的，也可以成为多才多艺的人”，我是这样合理化自己的，但这好像是很严重的错觉。随着时间流逝，我仔细地想了想，那个想法本身是因为我没有对自己深深地扎根。比起规定自己擅长什么、不擅长什么，我希望能够成为深深扎根的大人，所以现在我还是无法自信地介绍自己是演员崔秀英、歌手秀英。因为我内心里仍在扎根中，我认为如果扎根的话，不管做不做得好，都会产生那份真诚。

-请向《Allure Korea》的读者提问吧，这是为了通过给很多生活均衡的人具体的话题来给他们思考的时间。

（大家）不是随生活去思考，而是随心所欲地活着吗？
