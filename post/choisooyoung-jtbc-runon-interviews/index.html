采访翻译｜崔秀英2021年JTBC水木剧《Run On》终映采访

2021-02-08 17 min read # 奔向爱情 Run On # 电视剧 Drama # 秀英 Sooyoung # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

在JTBC水木电视剧《Run On》中饰演徐丹雅一角的崔秀英2月8日早上接受了视频采访，传达了电视剧幕后发生的故事。
《Run On》讲述在用韩语来进行交流都困难的时代，每个人以各自不同的语言、不同的速度朝着彼此的方向完成奔跑的爱情电视剧。

秀英所饰演的徐丹雅是一个“做不到的事只是因为没有去做的女人”，既是体育经纪公司的代表又是大企业的常务，拥有这样华丽事业的丹雅并不满足于天生的富足，而是追求某种成就感的积极进取的人物。
秀英变身成为追求完美的事业女性徐丹雅，和李映禾（姜泰伍 饰演）的爱情、和吴薇朱（申世景）的友情等，她细腻地表现了和不同人物之间的化学反应，因此获得好评。

-有什么选角的插曲吗？（news1）

我在阅读剧本的时候，曾经想过这个角色是不是更适合性格比我更冷酷一点的演员、或年龄比我大一点的演员，也想过（制作组）是觉得我的哪一面是符合这个角色的，但是给予我至今从未展现过形象的角色这一提议本身就让我觉得很兴奋，觉得是不是制作组看到了我自己都没看到一面。编剧说过她是我的粉丝，我还觉得可能只是客套的问候，并没有完全相信，但她却看过我之前所有的作品，似乎也了解我当下是以什么样的心境去表演。我觉得编剧对演员秀英的喜爱让我更感动，她告诉我徐丹雅这个角色除了秀英没有别人能演的时候，我觉得不是因为在剧本中能刚好想起来的演员，而是想尝试与众不同的作品，因此决定出演。

-《Run On》具有人气的秘诀是什么？（news1）

电视剧里讲述工作出色的女性、年轻世代的青春和爱情的时候，这是不是以“最近的年轻人是这样的吧”来衡量进行制作的，我曾经有过这样的想法。但是我觉得《Run On》反而是让大人们产生“最近的年轻人真的是那样吗？”这种想法，很好地融入了那一代人的苦恼和思考的电视剧。不是站在远处猜测着他们的人生而写出来的电视剧，而是编剧和我是同一代人，而且和我们度过了同样时代的青春才写出的作品。我觉得演员们都很准确地了解这一点，并以此去表演，因此化学反应很不错。

-《Run On》中值得回味的台词成为了话题，给你印象最深刻的台词是什么呢？（news1）

有很多有趣的台词，比如“你的脑袋是装饰吗”、薇朱的“有病”这样骂着说出的台词我很喜欢。我们电视剧的另一种魅力就是既有趣又让人难舍难分。结尾有句台词是“如果你相信我的话，我也会相信你”，我看的时候哭得很厉害，因为这部电视剧对我来说就是那样的存在。我在这部作品中倾注的热情，有时人们也会像工作一样冷静、理所当然地接受，这是一部能够切实地感受到反馈的作品。虽然是有所不足的角色，但我能够完全相信地去进行演绎，是因为相信编剧能够解开丹雅的故事线。以“Run On”来展现“我会成为你的，只要你相信我，我就能做到”，这和我很像。这是一部作为一个人、作为角色都能得到安慰的作品，我在想那句话是不是能完全解释《Run On》的台词，所以至今我还记得。

-徐丹雅的魅力是什么？（Hearld POP）

其实在电视剧中也有很多涉及到了女性领袖角色，但像徐丹雅这样年纪轻轻就成为女性领袖的角色似乎并不多见。我也觉得很有负担，也希望以后会出现更多这样的角色。让人觉得她“不只是有位置，也很有实力”，我集中精力去展现她是一个工作出色的女性，似乎是这一点成为她的魅力而获得了观众们的喜爱。

-关于台词“只要你相信我，我就会成为做到的那个人”（OSEN）

徐丹雅这样有点缺乏的角色能否连成长也能完结，难免会有这样的不安。从初期开始，以不管是谁都能获得好感的角色来设定，对我来说是轻松、安全的选择。但是能够出演有缺乏的角色，是因为对编剧有完全的信任才有可能的。即便看起来有点无礼，我也决心要出演这样有着成长和缺乏的角色。我相信编剧能够解开丹雅的故事线。大家看出我是带着信任去表演的，所以我作为一个人、作为角色本身也获得了慰藉。我想那一句台词是不是可以把《Run On》这部作品完全说明的台词呢？那句台词现在还留在我的心里。

-你和徐丹雅相同的地方、不同的地方是什么呢？（news1）

和丹雅相同的地方，丹雅和我都是希望能够完美地处理事情？从为了做好而去努力这一点来说，我们应该有70%程度的相像。丹雅遇到事情不顺的时候，无法调节自己的愤怒，而我是可以忍耐地很好，这一点我们是有区别的。另外，我也有过希望像徐丹雅一样想说什么就说什么的想法，通过在这次电视剧中饰演丹雅，好像消除了那样的心情，感觉很舒畅。我也觉得丹雅像少女时代，看上去是一开始就获得喜爱的人、拥有一切、看起来很完美，但其实为了准备得完美无缺，像是被追逐着一样生活，即便只有一点点时间，也会用来努力地自我管理。其实是为了达到别人眼中满足的标准，孤军奋斗地生活，这就像是在看着过去活动时的成员们。在这一方面，我是可以理解丹雅的。

-在徐丹雅和李映禾不现实的关系中，结尾太现实了。和李映禾的关系中，徐丹雅是什么样的心境？（news1）

升职近在眼前了，实现目标的机会出现在眼前，她似乎是觉得和李映禾的感情其实没什么大不了的。办葬礼的时候，她不能和映禾直接地见面。以“（映禾）就像是仲夏夜之梦一样的存在”来对待，我觉得这就像是一个失魂落魄的人的台词。把映禾丢在身后自己往前走，映禾却突然找来说着“这完完全全是只为了你的决定，不是因为爱你而放弃，而是因为爱你作下的决定”这样表达心意的台词。剧本上写着“在映禾的怀抱里，像孩子一样哭着的丹雅”，丹雅也知道在这个人面前她才能成为完整的自己，为了轻松地离别，对于微不足道的爱情的罪责感，就算我负心也依然爱我的大人般的存在？看着那样无限大的爱，和自己形成对比的样子，她像孩子一样，卸下武装之后的感情爆发。

-和申世景的合作如何？（news1）

我一直都很尊敬申世景这个演员，她是比我有着更多作品的前辈，也是从童年出道活跃到现在的女性艺人，尊敬的同时又觉得她很了不起。很感谢的是我和世景有很多有趣的戏份，第一次合作演戏能够配合得这么好是我没想到的（笑）。一起演戏的时候发现她尽管有着自己想要的准确的闪光点，但也不失灵活性。注意力非常集中，但也不忘观察四周。我在现场观察世景的时候，学到了很多。吴薇朱可以称得上是“人生角色”，而饰演吴薇朱这一角色的能够是世景，我作为观众觉得非常满足和尊敬。另外，她也很漂亮和善良，是我想一直为她鼓掌的演员。

关于申世景（MBC演艺）

她是我本来就非常喜欢的演员，也是同龄的朋友和大学同期。申世景非常喜欢我的幽默，也接受了很多我的点子。

关于“吴薇朱”（YTN）

薇朱特别地有自尊心，这让我很有印象。丹雅的台词中有一句“看起来没什么后台，但却不跪下。故事看起来很多，但却不悲伤。”这好像很能表现薇朱这个角色。我希望拥有的青春模样就是薇朱那样的。因为那些样子和我的青春有些相似，丹雅和薇朱都让我想起了我的青春。

关于申世景（YTN）

世景无论我说什么都咯咯咯地笑，从某个瞬间开始我就带着惹世景发笑地使命感去拍摄现场。和世景拍摄的日子，情绪非常高涨，觉得是让人兴奋起来的工作。在第一次和薇朱碰面的戏里，丹雅有句台词是“是议员塞过来的吴美子女士吗？”这是即兴发挥的。我有这种想法的话，对手演员会怎么想，没有这样的不安和负担才能自由自在地沟通，不管什么点子都毫不犹豫地成为我所创造的，那样的关怀和度量让我很尊敬。

-现在看到在《Run On》中是姜泰伍、在《新年前夜》中是刘台午，你遇到了两个“Tae-oh”。（news1）

刘台午演员是进行准备后再表演，因此是有很多细节点的演员，也带有他特有的气质，很有魅力。如果下次能够合作，我希望我是饰演强势的角色。姜泰伍演员则是我第一次和年下男设定的演员演对手戏，泰伍是第一次演双方的爱情戏，所以我特别有负担，因为希望可以留下美好的爱情、希望看起来美好的这些想法而感到负担。演着演着觉得很惊讶，因为他是后辈、年纪又比我小，我做了些多余了干预，但他本来性格很好而全盘接受了。和他说的时候，他全都能理解，而且消化得很好。是很聪明的朋友，并不是说我很聪明，而是他是一个很会观察周围、很有礼貌、很体贴的朋友。年纪小做得这么好并不容易，但我觉得他做得很好。

-关于姜泰伍（Star News）

他很懂照顾别人，对同僚演员也是充满了礼貌。泰伍也是入行7年的演员了，我觉得他成功是有原因的。我觉得自己好像是很有对手演员福气的演员。

-关于姜泰伍（MBC演艺）

实际上本人的性格非常有男子气概，多亏他天生的男性气质，爱情戏演起来很容易接近。和姜泰伍一起演的年下男设定是我第一次尝试，据说也是姜泰伍出道后第一次表演双方爱情戏，所以我还蛮有负担的，因为要把姜泰伍这样有魅力的演员展现给大家才行。

-和任时完的合作如何？（news1）

和时完欧巴一起演戏真的学到了很多，我在剧本里看到的奇善谦和实际任时完演出的奇善谦诞生时，我是觉得惊叹和起鸡皮疙瘩的。和时完欧巴一起合作，感受到了他特别地自信，那份自信我觉得是对这个角色进行了多少的研究才能产生的确信。在现场，他没有犹豫、没有怀疑，确信里又特别自信。

-你和任时完同样都是爱豆出身，有没有形成共鸣或彼此依靠的部分？（news1）

现在来看我们当时都没有那样的想法，没有想过是同僚歌手。时完欧巴本来就比较喜欢聊关于角色本身、电视剧和其他角色的事，所以才没想起是爱豆吧（笑）。

-关于任时完（Star News）

很想展现我所感受到的丹雅和善谦的同质感，和善谦的对话中似乎出现了奇妙的张力，连我都觉得起鸡皮疙瘩和喜欢。

-少女时代成员们的反应如何？允儿也在同时期播出的《Hush》里出演了。（news1）

一般开始播出的时候我们会在群里聊起来，这次我们两个在相同的时期出演作品，因此不能为彼此仔细地监督。成员们总是在彼此有作品的时候互相联系加油打气。

-关于少女时代成员（体育朝鲜）

我们本来在作品开始的时候，会在群里和大家打招呼。这次和允儿的电视剧是同时期播出，因此我们不能互相好好地收看对方的作品。但是我看了《Hush》的首播（笑），允儿也说‘看了《Run On》’给我应援，我们好像都是同样的心情，因为作品在同一时期不能仔细收看彼此的作品而感到抱歉，允儿似乎也是这样。成员们是最彼此互相支持的。

-少女时代成员们的反馈（YTN）

成员们为丹雅和映禾的爱情线应援，作为热烈的观众在收看着。像是Yuri、徐贤这样也在进行演员活动的成员，丹雅这样痛快的角色是我来饰演的让她们觉得很痛快，尤其是目前为止少女时代中还没有人像丹雅这样有钱的角色，成员们说‘秀英在有钱人里漂亮地出来了，真的太好了’。Tiffany特别认真地帮我检查，丹雅在济州岛的英文台词其实因为Tiffany的翻译才全部完成的，本来是韩文的台词，她把角色的特性和要说的重点很好地融合，翻译成英文台词。我在这里很想和她说真的谢谢。有这样可以给我建议和点子、毫无顾虑地交流意见的朋友和同僚，对我来说是很大的财富。

-现实生活中崔秀英工作和恋爱时是什么样子的？（news1）

我没有想过把工作和爱情区分来看待，而是把两者都看作是我人生中的一部分。不知道是不是因为最近的作品，很多人问我“工作和爱情会选择哪一个”，没有区分两者，而是根据当时最优先考虑的事情去转换，然后尽全力地去完成。我没有想过必须优先去做哪一个，对于我所得到的工作、所获得的爱，我总是竭尽全力，这好像就是我的使命和自豪感。

-对于爱情戏的演技，郑敬淏的反应如何？（news1）

我认为那位是大韩民国真的非常擅长爱情喜剧的演员和前辈，那样的演员一直站在我可以经常提问的位置，对我来说是一种祝福。我问了很多问题，也得到了很多帮助。

-有没有为了看起来更像有钱人而努力？（news1）

即使没有努力看起来像有钱人，设定本身也是个有钱人。我有想过比这次所展现的着装更平凡的，比如穿着运动服、拿着环保包出门的女性。拍摄电视剧的时候，制作组的意见是因为是在时尚界工作的女人，所以一出场给人是时尚界从业者的印象更好，结果就稍微华丽了一些。我想过避免让丹雅这一角色在电视剧中变成目中无人的财阀女，希望可以展现她是自我管理的终结者、工作出色的事业女性。想象在电视剧中的财阀角色时，不是都会有自动联想的形象吗？所以我希望她的形象不是活在自己世界里的公主，而是有着自己的理念和思想的女性。

-关于徐丹雅的角色（体育朝鲜）

我希望能够避免让她成为目中无人的财阀继承女，这也不是编剧希望的方向，希望能展现她是自我管理的终结者、工作出色的事业女性。大家想象财阀电视剧时会有自动联想的角色，我希望她不是活在自己世界里的公主，而是理念思想被唤醒、具有现实性的人物。所以丹雅为了保护环境而随身携带着保温杯，做了这种独属于我的设定，那样的小道具瞬间增加了不少时尚感，感觉更好地塑造了角色。如果把丹雅塑造成比较普遍的角色，会很安全，但没有那样做。说话在剧本里总是以'~했니'来结束，让我觉得负担，我去问编剧的时候，她希望我一定要那样说话，所以我干净利落地整理了（我的负担）。

-在对徐丹雅进行设定时的着重点（MBC演艺）

我想象过她是穿着特别舒适的衣服、扎着马尾、拿着平板电脑坐在飞机头等舱的女人，也想象过她是虽然穿着休闲但却有个性、不在意别人眼光、专注的女人，也不背名牌包。说话并不是无理的语气，而是希望表现出她是有理由的、堂堂正正的，不是有着那种自尊心很强的人吗。导演看着我的表演说他感觉我是“同龄人无法匹敌的疯子”，能够如我所研究和设定的那样获得反馈，让我觉得更高兴的作品。

-似乎作为演员已经站稳脚跟了，你有觉得自己这份工作做对了吗？（news1）

那种成就似乎非常大。我觉得自己在这个行业已经呆了很久了，都快有20年了，真的看到过很多类型的人，而我心中的门曾经关上过、也打开过，还假装那样过。在做这份工作的时候，为了不受伤害，经历过很多曲折，但还是觉得自己的等待很值得。作为艺人秀英，我有过因信任而受伤的经历，也积累了害怕受伤而无法完全投入热情的经历。如此下来虽然会有些怀疑，但遇到这样无限温暖和无害的电视剧，遇到欣然报答演员在作品里竭尽全力的那份热情和爱的制作组，我觉得这份工作做对了。
