采访翻译 | 徐贤2018年MBC电视剧《时间》终映采访（日刊体育）

2019-05-31 12 min read # 电视剧 Drama # 徐贤 Seohyun # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

演员徐贤通过MBC水木剧《时间》夯实了迷你系列剧主人公的基础。从一开始就出现很多闲话，一起合作的演员金正铉在制作发表会时拒绝和徐贤挽手、一直维持无表情的状态让他的态度争议扩散开来，之后金正铉又因为健康的理由，在电视剧结束前中途下车。徐贤需要填满所有男主人公突然消失留下的空白，主人公的分量无疑只能变得更大，承受那份分量并坚持到底成为了徐贤的课题。尽管多少是有些毛糙的申告式，但还算做得不错，在“徐贤的再发现”这样的评价中画下了句号。

- 电视剧终映后是怎么度过的？

大概3天前才从薛智贤这个角色中脱离出来，因为是消耗了很多感情的角色，终映后身体不舒服躺了一周，好像因为紧张的情绪一下子放开而病倒了。

- 得到了“再发现”这样的评价。

表演虽然是假的，但也是将假的东西用真实的感情去表现的职业。“该怎么做才能每一刻都能把假的东西真实地去表现”这是一个非常大的课题。过去一直同时并行着几个活动，不能全身心地只集中在表演上，觉得很遗憾。因为很渴望（只集中演戏），所以这次拍戏的时候，别的事什么都不做，只集中在这部作品上。连日常的生活都配合着作品过着，虽然我设置了一个概要，但还是特别困难。当时觉得只是普通地努力好像不行，所以为了能够专心地和薛智贤交流，我希望能有只属于自己地时间，因此在拍摄电视剧期间，我和父母分开生活了。

- 觉得最为自豪的评价是什么？

在最后的拍摄阶段，几乎像小剧本一样出现。我们有一场戏是进行直播拍摄，在街上走着的时候，通过电话来揭开真相，那是作品中最为重要的一个场面。在拍摄开始1小时前剧本才出来，有8页纸的拍摄份量，因为是前一天的剧本完全不同，让我觉得有点“崩溃”。当时在想我该怎么做才行，觉得如果在这里出错的话，这部作品就可能要完蛋了，因此我心理上的负担很重。当安全地完成拍摄时，我真的觉得很自豪，也去和导演说了谢谢。是让人觉得惊险的瞬间。

- 这是一部有很多惊险瞬间的作品。

在作品内外都有很多让人精神动摇的因素，得益于此，我的精神力也变得如钢铁一般。什么事我都不会害怕。好像这也是没办法的事，那些状况不是我想去避开就能避开的，我觉得自己真的很好地去经历了。在那样的状况下依然收看这部作品的观众们，很感谢你们。

- 选择这部作品的理由是什么呢？

想起徐贤这个演员的时候，比起合适的角色，我觉得虽然没有印象，但很好地消化后受到的冲击应该更大，所以我想挑战看看。和张俊浩PD一起合作过上一部作品《小偷家伙，小偷大人》，虽然是导演执导的作品，但他想和我一起合作所以联系了我，我也希望可以报答这份信任，所以想做的更好。作品所要传达的讯息也很深刻，所有人相同的时间里却过着不同的人生，我对这样的讯息产生了共鸣。

- 作为第一部迷你剧主人公的作品，有没有觉得负担？

不管拍摄哪部作品都会觉得负担，这个人物是由我来创造的人物，所以在表现这个人物的人生时，必须得负起责任才行。在这部作品里的人物的人生过得很痛苦，所以更不容易了，是我从未经历过事，我得理解人物后和人物融为一体变得非常重要。

- 金正铉在中途下车了。

不管怎么说，我只能背负起更大的责任。如果我在这里动摇的话或是犯错的话，我觉得作品可能会完蛋，所以我不能让那样的可能性发生，我只想到了“无条件要好好去做”、“以后得走下去”。虽然假装若无其事，但我心里其实很害怕，身边的工作人员和演员们都理解我这样的心情，都为我加油打气。多亏了那样的氛围，才能很好地结束。

- 酒醉的那段表演很真实。

看起来不会有点疯狂吗？（笑）其实我喝酒的时候是没有喝到最后过的，所以也没有人见过我喝醉的样子，我原本也很讨厌那样。我本身并不想让别人看到我崩坏的样子，但这次应该说是通过作品表现了自我。导演的目标是“像疯子一样”，所以为了表现少珠喝了一杯之后感情起伏很大，我真的像疯子一样去表演了。

- 饰演前男友的金俊翰实际上是怎样的人呢？

欧巴性格和申珉硕这个角色相反，是个很有趣的人。他和角色不相符的性格让我很吃惊，虽然我们之间有着明显的年龄差，但相处得很舒服，也开着玩笑，演戏时很专注，很好地传达和接收了。有趣地拍摄了。因为拍摄时很沉重，所以休息时努力地开朗。因为演员们本身的性格都很开朗，拍摄现场没有变得很阴郁。

- 据说演完阴郁的人物后会觉得很辛苦。

5个月期间没有和身边的朋友好好见面，只想着和角色相关的事情。一直带着角色的感情是最辛苦的，亲近的朋友们虽然叫了我出门，但在朋友们看来我看上去真的太阴郁了。因为是第一次看见我这样，还担心地说“作品拍完了，是不是需要接受精神科治疗才行”。想着和作品相关的事情时，我是没法笑出来的，有太多融入了我的生活中。

- 成为最大力量来源的存在是什么呢？

名字叫做bobo的小狗成为了我的力量，如果没有它的话可能要出大事了（笑）。虽然它不能说话，但给它多少爱就能从它那里得到多少爱，成为很大的慰藉。就那样看着它也觉得幸福，看着看着也会自己流泪，给我慰藉，给我治愈。我和小狗拍了广告。

- 少女时代成员们的反应如何？

欧尼们送来了咖啡车、零食车。因为作品困难，欧尼们都很担心。孝渊欧尼亲自找到现场来了，我一见到欧尼，眼泪疯狂地流了出来，感觉就像很久没有和妈妈见面一样，解除了武装的感觉。以前虽然一直觉得像家人一样，但最近这样的感觉更强烈了，我们分开着进行活动下来，更感觉到欧尼们的存在是特别珍贵的，更想念她们了。拍摄作品的过程中，也是我很大的力量。

- 和少女时代成员们见面的话，最喜欢的地方是什么？

爆发“比格美”，像回到10代那时的感觉。比起聊严肃的事情，我们更喜欢聊有趣的事。身边有着这样的人们，我觉得非常地感谢。我们是变成老奶奶时还能继续见面的关系。

- 保持超过10年的友情的秘诀是什么？

我们之间经常聊天。每天睡之前必须拿出5分钟和彼此聊天，对彼此有伤感情的地方，聊着聊着也就解开了。我们之间虽然也会吵架，但彼此理解的时也有着帮助。各自过着不同人生的人们，一起生活5年以上不是那么容易的，是可以向彼此学习的一段时光。

- Yuri开始Solo挑战了。

预告照都非常漂亮，可以说是“取向狙击”，少女时代成员们都为她应援。就像是第一次挑战Solo那样，决心好像也很不同。这不正是可以展示独属于欧尼的色彩的机会吗？我想快点听到歌曲，很期待。

- 和朝鲜艺术团一起登上了首尔公演的舞台。

是我人生中无法重来的经历。我是首尔公演开始当天收到了联系，非常感激地接受了提议。一开始我还有点不知所措，想着“为什么联系了我呢？”我还怀疑是不是电话诈骗呢（笑）。我和父母说这事的时候，仔细想了想，能够联系我这本身就是对我的信任，因此觉得要好好完成才行。虽然觉得负担，但下定决心做了，就开始表演准备工作，以“我们国家的代表”这样的想法去准备。所幸的是没有发生什么问题。我当时心里是无比地紧张，表演正式开始后，作为少女时代活动时的舞台经历发挥了作用，我没有所担心的那么紧张了。和北韩的歌手牵着手一起演唱、交流，有种进入电影里的感觉，像做梦一样。

- 之后在北韩的演出也以MC的身份一起进行了。

因为是让我担当重任，那时我也问了“这让我来做也是可以的吗？”去平壤的那几天，是我病得最厉害的时候，高烧到了40度都没有降下来，身体也变得不舒服，24小时里一直在咳嗽，连声音都变哑了，甚至还发不出声。彩排的时候，身边的人一直担心“能做到吗？”我和工作人员们聚到一起祈祷着。奇迹地，登上舞台的时候声音出来了，所以结束时我哭了，庆幸自己很好地完成了，回到宿舍我几乎都要晕倒了。

- 尝了北韩的平壤冷面吗？

吃是吃了，但因为我当时感冒了，也不知道留在记忆中的味道是不是对的。它没有刺激性的味道，像面条一样美味的冷面，是之前没尝过的味道。

- 有和一起去平壤公演的前辈们联系吗？

我们有团体SNS群，前辈们鼓励我：“这些不是轻易能经历的，因为是一起才能做到。”赵容弼前辈虽然也在群里，但基本不怎么发言。前辈们对我很亲切，也常常为我加油，让我感觉有了新的家人们。

- 离开SM娱乐已经超过一年了。

包括练习生时期，在SM的时间有15年之长，我很骄傲自己像是“金汤匙”一样。虽然在那里也有觉得辛苦的事情，但和其他环境比较的时候，真的是一个非常好的环境。很感谢把我养育到现在位置的公司，但我想自己去碰撞着看看，有着想去创造的渴望。虽然现在还没找到正确的答案，但我觉得这是寻找的过程。

- 有继续独自进行工作的想法吗？

拍戏的时候，就算只是集中在演戏上都有点吃力，还要把心思花费在其他事情上并不容易。所以，虽然现在不能马上决定要怎么选择，但感觉到了公司的必要性。和一些公司也进行了会面。

- 也可以看到歌手徐贤吗？

虽然想首先集中在演戏这方面，但我的老本行是歌手嘛，总有一天希望能够让大家听到我的音乐，歌曲也在一点点地写着，虽然不知道会是什么时候，也不知道会是什么样的歌曲。

- 你有着正直生活的形象。

好像是通过综艺固定了大家对我的印象，虽然大众是通过综艺看到我的模样，但那好像静止在了10年前，那时的样子好像更强烈、更突出。但随着时间的流逝，人是会变的，也会有更多的样子。我并不想强行去打破那样的形象，因为那也是我的样子，我也还有其他的样子，因此想更多样化地去接受。

- 没有出演综艺节目的计划吗？

似乎有必要更亲近大众，作品开拍前，听到了很多出演真人秀综艺的提议，当时我想等作品结束后再进行，我想慢慢来。也有收到MBC《我独自生活》的出演提议，但情况不顺利所以没能成功出演。况且，我现在是和父母一起生活，以后我一个人住的时候，一定要尝试一下。

- 将来有什么计划？

11月开始将会进行见面会，因为是第一次一个人来做这件事，所以很认真地在准备。下一部作品也很慎重地在挑选，主要是在看浪漫喜剧这一类比较开朗的类型。如果有机会的话，我想尝试类似缺心眼的或是四次元的角色。
