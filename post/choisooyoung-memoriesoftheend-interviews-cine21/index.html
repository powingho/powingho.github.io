采访翻译 | 崔秀英2019年电影《尽头的回忆》宣传采访（CINE21）

2019-04-12 8 min read # 秀英 Sooyoung # 少女时代 Girls' Generation SNSD # 采访 Interview # 电影 Movie # 翻译 Translation

《尽头的回忆》崔秀英，经历了成长痛后找到了新的自己

-去年在釜山国际电影节上《尽头的回忆》第一次公开了，今年初也在日本正式上映。釜山和日本的反应如何呢？

在釜山还来不及去感受观众们的反应。踏上红毯和以我的电影站在观众面前都是第一次，所以很紧张。少女时代活动的时候虽然登上过许多大舞台，但这一切都是新的经历，觉得很感动。在日本上映的时候，直接去买了电影票，偷偷地潜在观众中间看了电影。不管怎么说，在日本更多的是作为歌手被大家熟悉，慢慢地，如果有更多作为演员和大家接近的机会就好了。

-出演这部作品的理由是怎样的呢？

通过朋友知道了吉本芭娜娜的小说将会拍成电影，也收到了剧本。虽然是先看了小说原作，觉得说的是和我同龄的女性的故事，如果现在不拍的话，以后就拍不了的故事。正好那时我也经历了和电影情绪相似的成长痛，再加上是韩日合拍的电影，会在两国都上映，觉得是一次很好的机会，也是学习之路，是一部我不想错过的作品。后来和导演见了面，导演说我出现在了她的梦里，说是开始并没有想到崔秀英这个演员，但在梦里，我走进了导演用的雨伞里。就像《狼的诱惑》里的一个场面。（笑）

-这是一部集合女性原作者、女性制作人、女性导演、女性主人公的电影。由女性们来讲述关于女性的故事，决定出演的时候有没有受到这一点的影响？

一开始并没有什么特别的想法，（只是觉得）是改编自喜欢的作家的作品，制作公司代表想让女导演的执导，我出现在导演的梦里（这样而已）。（笑）直到制作开始后，我才意识到参与制作这部电影的许多人都是女性，这似乎也符合时代的潮流，成为有意义的一次挑战。

-原作小说里的主人公美美是日本女性，而电影里的宥美是韩国女性，在参考原作中的角色时应该有些困难吧？

原作中的美美有着被动的一面，等待着男友先来联系，也不会轻易怀疑男友。虽然考虑到这是10年前写的小说，但看到了主人公是至高至纯的女性的一面。如果是当今时代的女性的话，应该要说些什么，长时间没有联系的话至少会发生一次冲突。由日本人换成韩国人，然后我来饰演宥美，想着要不要增加一些积极的女性形象。演戏的时候，虽然很多时候会想着’如果是宥美的话，会怎么做’，但也有很多时候会想’如果是我，会怎么做’，也和导演谈了很多这样的看法。另一面，原作中是这程度的话要跟着走才行，有这样想法的宥美是个很好心的人。西山（田中俊介 饰）不是说过这样的话吗？’我知道你是一个内心很深、很成熟的朋友，但和你交往了4年的男友好像并不了解你，你依然是个很好心的人。’那时宥美感受到了没有错误地活着的安全感。

-宥美善良、成熟的另一面，看起来也像是长得漂亮却不知道困苦的人物一样。

3年前的我也和宥美一样，实话说那时候连银行都不知道怎么自己去。（笑）好像3、4年前有了想让自己置身到新的环境里的想法，离开工作了很久的公司，加入新的经纪公司，也是因为想在新的环境里历练自己，经历成长痛。虽然是没遇到什么困难，也没有其他问题过得很好的人，但在新的环境里暴露自己，也找到了另一个自我。有了新的勇气、有了挑战的意识，也开始认识新的朋友。最近可能是因为经历过那样的成长痛，所以想着是不是可以充分地理解宥美。

-知道男友有了其他女人后，宥美在民宿“End Point”住了几天。宥美和民宿的主人西山有没有发展成恋人的关系也让人觉得很有趣，怎么理解这两人的关系？

书里面（两人之间）是有着更微妙的的感情，但是这部电影更接近一个女人的成长期。虽然是讲述经历意想不到的离别后治愈伤痛的过程，但希望不要看起来是以新的爱情来治愈因爱情带来的伤痛，我和导演分享了这样的看法，希望是强调宥美不是因为遇到其他男人而治愈了伤痛，而是通过她自己得以治愈。西山只是将把自己关在房间里的宥美带出外面的人，仅此而已。在宥美和西山的眼神交流中，也努力避免那种甜蜜的感情氛围。

-和饰演西山的日本演员田中俊介以日语进行合作的过程中，有没有感到困难？

10代的时候有过在日本生活了3年的经历，因此用日语交流或是日文台词并不困难。但在把韩文台词翻译到日文的过程中，因为有一些不自然的台词，把心思都放在把那些台词如何自然地转换上。在日本拍摄前，花费最长时间的工作就是自然又恰当地修改日文台词。日文小说、韩文小说、韩文剧本、日文剧本，就这样拿着四本书和田中俊介对台词。找找这句台词在小说里是怎样的，再翻着文法书、句法书，大部分时间都用在寻找最自然的台词上。因此，好像和同龄的田中俊介变得更亲近了。

-和许多年龄相仿的朋友们同甘共苦超过10年，分享友情又彼此竞争地进行少女时代的活动。在（少女时代）里，该如何定位秀英的存在的苦恼好像一直都很多。

（我）没受到什么压力，我想到（这种问题）的时候，觉得自己是没有竞争力的角色。（笑）那时没有像我这样个子高的朋友，还有看起来很成熟的脸蛋。作为偶像，无论是外貌还是实力，我都不是有竞争力的角色。但因为我非常喜欢跳舞，非常喜欢少女时代，所以在舞台上也很享受。在（少女时代）里面，一边寻找我该做的事，一边享受着。没有过’这次该做什么发型？’’要比她更漂亮才行’那样的想法。少女时代的成员，与其说是在成员之间展开竞争，不如说是大家为了自身不断地苦恼和研究。虽然是成员和朋友，但作为同龄女性，她们很好地塑造了具有标志性的角色，这让我觉得很尊敬。我现在才刚刚开始。（笑）过去所经历的，我能做到的，现在开始想进行不同的尝试。少女时代活动的时候，基本上给我的压力是1/8，而现在我必须要独自承担所有的压力。

-希望自己的30代是什么模样？

我30代的时候和20代一样就好了。20代时应该有的各种苦恼和挑战，30代时也想去做。以前，我把30代想成是完成型的，但真正到了30岁才知道要去争斗的东西还是和以前一样很多，还是要去孤军奋斗。’好吧，我还在青春里’，这样想着心情就放松了很多。我想继续挑战下去。
