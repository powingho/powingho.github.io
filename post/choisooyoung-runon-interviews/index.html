采访翻译｜崔秀英2021年JTBC电视剧《Run On》终映采访（综合）

2021-03-14 14 min read # 新年前夜 New Year Blues # 奔向爱情 Run On # 电视剧 Drama # 秀英 Sooyoung # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

#关于徐丹雅

关于饰演徐丹雅（Tenasia）

我一直觉得我能够演好徐丹雅这样的角色，即使我之前没有通过作品展现那种模样，导演和编剧仍然相信并选择我，让我觉得神奇又感谢。有点感觉像是被人知道了只有我自己知道的愿望。

很久之前就期待遇到徐丹雅这样的角色（YTN）

各自都有着自己的长处，不管是还没有被发现的，还是为了将来有着藏得严严实实的武器。丹雅这样的角色让我来演的话，我觉得自己是可以演好的。给予我信任并选择我，让我觉得神奇又感谢。像是知道了我的愿望，我怀着激动的心情选择了。

在强者面前变得更杀气腾腾的“痛快寸铁杀人”让许多观众患上了“徐丹雅病”（YTN）

表演的时候我没有可以变现出痛快感和刺激感，因为是丹雅这个角色有可能会说的台词，所以看剧本时没有异样的感觉。徐丹雅这个角色原本就有很多演技上的趣味，我一直都相信编剧不会让“角色崩塌”，而是让丹雅得到成长。即便不是我来出演，我相信观众们也会爱上徐丹雅的。

徐丹雅和之前电视剧中财阀继承女有什么不同？（YTN）

她不是之前电视剧中所看到的财阀家女儿，这是为了展现自主性、主动性的设定。随身带着保温杯出门怎么样？即使没有如此，为了北极熊举行活动的设定据说也是早就想好的。应该说是充满正义感的有钱人吗？因为是理念和思想都觉醒的朋友，即使没有以场面来表现，能不能在视觉上表现出来，做到的就是保温杯。运动鞋是编剧写的设定，我有见过在纽约慢跑的女人，穿着秘书给的相差百万的外套直接去上班的样子。我觉得对丹雅来说那个样子是非常适合的。

徐丹雅也不是完美的人物（YTN）

事实上有一些背离感。徐丹雅的台词很厉害，让我觉得“现实中会有这样说话的人吗？”如果我觉得丹雅的台词无礼的瞬间，她就成了明知道对方感情却不体谅、目中无人的继承女角色；或是我觉得丹雅的语气是成长过程中因为缺乏而出现的必然反应，这就成为她另一个成长的跳板。最后我有多相信似乎就成为了（这两者的）差异。剧本和现实之间背离的差距越演变得越小，这难道不是意味着剧本写得很好吗？在捕捉角色发展过程中的其中一个过程时，可以切实感受到编剧的宏伟蓝图。站在演员的立场，是非常感谢的剧本。

和徐丹雅的成长一起，崔秀英的演技也进一步成熟了（YTN）

丹雅的成长似乎有着很大的意义。丹雅和睿俊道歉的时候，她变成了一个也会道歉的人。虽然微小，但却是很大大变化。稍微有点学会了看别人的眼色，也学会了照顾对方的感情。该说是社会性吗？她通过李映禾这个人，她好像一个个学会了女性领袖必须具备的品德中的爱、关怀、施舍这些东西，只是还没有成为完成型。虽然电视剧结束了，但感觉她依然会带着不足，成为一边学习一边生活的丹雅。

-关于徐丹雅的角色（体育朝鲜）

我希望能够避免让她成为目中无人的财阀继承女，这也不是编剧希望的方向，希望能展现她是自我管理的终结者、工作出色的事业女性。大家想象财阀电视剧时会有自动联想的角色，我希望她不是活在自己世界里的公主，而是理念思想被唤醒、具有现实性的人物。所以丹雅为了保护环境而随身携带着保温杯，做了这种独属于我的设定，那样的小道具瞬间增加了不少时尚感，感觉更好地塑造了角色。如果把丹雅塑造成比较普遍的角色，会很安全，但没有那样做。说话在剧本里总是以'~했니'来结束，让我觉得负担，我去问编剧的时候，她希望我一定要那样说话，所以我干净利落地整理了（我的负担）。

-在对徐丹雅进行设定时的着重点（MBC演艺）

我想象过她是穿着特别舒适的衣服、扎着马尾、拿着平板电脑坐在飞机头等舱的女人，也想象过她是虽然穿着休闲但却有个性、不在意别人眼光、专注的女人，也不背名牌包。说话并不是无理的语气，而是希望表现出她是有理由的、堂堂正正的，不是有着那种自尊心很强的人吗。导演看着我的表演说他感觉我是“同龄人无法匹敌的疯子”，能够如我所研究和设定的那样获得反馈，让我觉得更高兴的作品。

-徐丹雅的魅力是什么？（Hearld POP）

其实在电视剧中也有很多涉及到了女性领袖角色，但像徐丹雅这样年纪轻轻就成为女性领袖的角色似乎并不多见。我也觉得很有负担，也希望以后会出现更多这样的角色。让人觉得她“不只是有位置，也很有实力”，我集中精力去展现她是一个工作出色的女性，似乎是这一点成为她的魅力而获得了观众们的喜爱。

-关于台词“只要你相信我，我就会成为做到的那个人”（OSEN）

徐丹雅这样有点缺乏的角色能否连成长也能完结，难免会有这样的不安。从初期开始，以不管是谁都能获得好感的角色来设定，对我来说是轻松、安全的选择。但是能够出演有缺乏的角色，是因为对编剧有完全的信任才有可能的。即便看起来有点无礼，我也决心要出演这样有着成长和缺乏的角色。我相信编剧能够解开丹雅的故事线。大家看出我是带着信任去表演的，所以我作为一个人、作为角色本身也获得了慰藉。我想那一句台词是不是可以把《Run On》这部作品完全说明的台词呢？那句台词现在还留在我的心里。

#关于李映禾

初期叫映禾“学生”的原因（Tenasia）

徐丹雅有个设定是不能很好记住名字，她总是只以职务来叫人。爸爸叫作会长、哥哥叫作徐专务。不能记住名字也意味着那个人谁都不关心，只是成就我的工作的人和手段而已，什么样的人并不重要。所以，刚开始见到李映禾的时候，他的职位就是学生，之后徐丹雅记住了映禾的名字，那是徐丹雅微小却很大的变化。

和映禾离别的场面（YTN）

映禾太像个大人了，丹雅像是卸下武装后流下的眼泪，就像是3岁的孩子一样。在妈妈的责备下，一方面她意识到自己的固执和孩子一样而感到羞愧，另一方面在想自己什么时候再会得到这样的爱，有着复杂感情的一场戏。

-关于姜泰伍（MBC演艺）

实际上本人的性格非常有男子气概，多亏他天生的男性气质，爱情戏演起来很容易接近。和姜泰伍一起演的年下男设定是我第一次尝试，据说也是姜泰伍出道后第一次表演双方爱情戏，所以我还蛮有负担的，因为要把姜泰伍这样有魅力的演员展现给大家才行。

-关于姜泰伍（Starnews）

他很懂照顾别人，对同僚演员也是充满了礼貌。泰伍也是入行7年的演员了，我觉得他成功是有原因的。我觉得自己好像是很有对手演员福气的演员。

#关于崔秀英

-关于任时完（Starnews）

很想展现我所感受到的丹雅和善谦的同质感，和善谦的对话中似乎出现了奇妙的张力，连我都觉得起鸡皮疙瘩和喜欢。

和自己的青春相似的角色是谁？（Tenasia）

徐丹雅和吴薇朱都有。薇朱的自尊感很强，我一直想变成的模样就是薇朱自尊感强的模样。看着徐丹雅，和少女时代活动时的样子很像，虽然看起来华丽、拥有了一切，但事实上特别奋斗地活着。为了追求完美，不得不无止尽地自我管理，特别希望休息的生活。那似乎像是我的青春。

和申世景的合作（YTN）

世景无论我说什么都是咯咯咯地笑，从某个瞬间开始我就带着惹世景发笑地使命感去拍摄现场。和世景拍摄的时候，情绪很高涨，是让人兴奋的工作。在第一次和薇朱碰面的戏里，丹雅有句台词是“是议员塞过来的吴美子女士吗？”这是即兴发挥的。我有这种想法的话，对手演员会怎么想，没有这样的不安和负担才能自由自在地沟通。不管什么点子都毫不犹豫地成为我所创造的，那样的关怀和度量让我很尊敬。

关于申世景（MBC演艺）

她是我本来就非常喜欢的演员，也是同龄的朋友和大学同期。申世景非常喜欢我的幽默，也接受了很多我的点子。

对吴薇朱和徐丹雅的喜爱（YTN）

薇朱特别地有自尊心，这让我很有印象。丹雅的台词中有一句“看起来没什么后台，但却不跪下。故事看起来很多，但却不悲伤。”这好像很能表现薇朱这个角色。我希望拥有的青春模样就是薇朱那样的。看着丹雅想起的是，她和少女时代很像，以前活动的时候，虽然看起来华丽、似乎拥有了一切，但其实都是孤军奋斗地活着不是吗？哪怕只有一点空闲，也要为了完美而自我管理，特别想歇下来的那种生活。因为那些样子和我的青春有些相似，丹雅和薇朱都让我想起了我的青春。

不知道机会什么时候会到来，一直穿着运动鞋的徐丹雅（YTN）

似乎也有导演认为起用秀英这个演员是冒险的，我觉得直到消除那样想法，现在还有很多的作业需要去完成。我也希望可以出演像丹雅那样角色形象坚强的人物，也希望可以出演成为主人公后从头到尾引领故事发展的讲述者的角色。但是，不管是什么角色，我都希望可以挑战不同的作品。

#关于少女时代

少女时代收看《Run On》的反应（Tenasia）

孝渊说她真的非常喜欢映禾的角色，深陷我的爱情线里，让她特别特别心动，是一个狂热的观众，说着“会变成怎样呢”在收看着。像Yuri和徐贤这样同样演戏的朋友则说徐丹雅的角色由同是少女时代成员的我来出演让她们觉得很痛快，我以一个有钱人漂亮地出现她们觉得太好了。Tiffany是最认真帮我检查的人，英语台词也都是Tiffany帮忙完成的。剧本里是韩文台词的，收到翻译之前我想提前练习而私下拜托了Tiffany，她把角色的特性和想说的重点都融入到了英文里。以Tiffany的构思，徐丹雅的英文台词诞生了。

- 少女时代成员们的反馈（YTN）

成员们为丹雅和映禾的爱情线应援，作为热烈的观众在收看着。像是Yuri、徐贤这样也在进行演员活动的成员，丹雅这样痛快的角色是我来饰演的让她们觉得很痛快，尤其是目前为止少女时代中还没有人像丹雅这样有钱的角色，成员们说‘秀英在有钱人里漂亮地出来了，真的太好了’。Tiffany特别认真地帮我检查，丹雅在济州岛的英文台词其实因为Tiffany的翻译才全部完成的，本来是韩文的台词，她把角色的特性和要说的重点很好地融合，翻译成英文台词。我在这里很想和她说真的谢谢。有这样可以给我建议和点子、毫无顾虑地交流意见的朋友和同僚，对我来说是很大的财富。

-关于少女时代成员（体育朝鲜）

我们本来在作品开始的时候，会在群里和大家打招呼。这次和允儿的电视剧是同时期播出，因此我们不能互相好好地收看对方的作品。但是我看了《Hush》的首播（笑），允儿也说‘看了《Run On》’给我应援，我们好像都是同样的心情，因为作品在同一时期不能仔细收看彼此的作品而感到抱歉，允儿似乎也是这样。成员们是最彼此互相支持的。

#关于电影《新年前夜》

对于与刘台午、姜泰伍的合作而以“爱情戏达人”重振雄风（体育京乡）

真是非常让人眼馋的修饰语，其实一听到这么说开心到有点堂皇了。（爱情戏是从和演员甘宇成合作开始的）我想像家训一样挂在家里的是“我和甘宇成前辈演过爱情戏”，想要子孙们世世代代传颂下去。（爱情戏的秘诀）不管是同龄演员们还是后辈们，在拍摄现场总是有很多值得向他们学习的地方，可以亲眼看到和我经历同一时期成长过来的的同僚演员。和说起是秘诀相比，我也在努力，在拍摄现场向其他演员们学习就是我的秘诀。（再一次强调）那开始的第一次就是甘宇成。

表达对电影《新年前夜》的喜爱（体育京乡）

值得感谢的是，因为再一次的新年，《新年前夜》才能够上映。因为是前年、去年拍摄的作品，我也是特别特别期待上映。作品本身就让人非常心动，所以像最近这样需要依恋的感性的时候去看更好。《新年前夜》中有很多出场人物，拍摄的时候我有负担要展现什么，幸运的是，洪智映导演使出了特别大的力气来帮助我。在导演看来，五月的故事也是这部电影的走向之一，所以她说不用那么费力也行。导演是看森林的人，有着准确考虑过的地方，我想要费劲心思完全使出演技的力量，但导演很好地调整了可以挽回五月可爱魅力的尺度。
