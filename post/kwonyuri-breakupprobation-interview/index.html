采访翻译｜权俞利2021年迷你电视剧《暂缓离别，一周》一问一答

2021-02-22 3 min read # 暂缓离别 # 俞利 Yuri # 电视剧 Drama # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

《暂缓离别，一周》是由权俞利、显祐主演的奇幻爱情电视剧，讲述在结婚前夕，遭遇意外事故离世的佳岚为了救活未婚夫善宰，要和他在一周内分手。2月22日至26日通过SBS Fil播出。

-决定出演《暂缓离别，一周》的理由是什么呢？

刚开始读剧本的时候，对20-30代平凡的男女爱情很有共鸣，最后结尾的时候感受到了强烈的回响。我希望可以通过佳岚传达这样的讯息，所以决定了出演。

-权俞利和剧中朴佳岚的相似度？

佳岚虽然平凡，但在爱情里有多喜欢就有多坦率，也有着爽快的性格。另外，在佳岚看到善宰为了守护她而努力的痕迹后，对爱情的态度变得更加成熟这样有魅力的人物。对每件事都坦率地表达感情，对我的人、我的东西不管发生什么都坚持到最后不会放弃，这和我蛮像的。不知道是不是因为这样，剧本越看越对佳岚这个人物产生共鸣。

-有没有留下印象的戏份？

我想起最后结尾时，看起来已经没有希望的善宰呼喊佳岚的场面。表演的时候也让我非常感动，结束拍摄后也是很难平复自己的情绪，留下了这样的记忆。

-在身边饰演的对手戏的显祐是怎样的演员呢？

我感受到他是一个真心关怀他人、温暖的人，每个场面我们都一起苦恼（如何表演），多亏了他我在拍摄现场真的很自在、很享受。

-和显祐的合作如何，如果给默契打分的话？

因为这次电视剧最重要的是和对手演员的默契，所以我和显祐在事前都会进行充分的讨论后再拍摄，默契分数我想给90分，剩下的10分我觉得由导演来填补。（笑）

-拍摄的时候，有没有觉得困难或辛苦的部分？

不管怎么说，因为大部分场面都是以佳岚的感情线为中心来讲述，为了能够更细致、充实地表现台词和行动，我苦恼了很多。与其说是觉得困难，我为了更集中于佳岚，付出了很多努力在拍摄以外的时间最小化地消耗感情。

-你参与了《暂缓离别，一周》的OST。

我对这部作品有着很深的感情，能够演唱主题曲《暂缓离别》这首OST我觉得很开心和幸福。特别是从第一次听到歌曲开始，我就喜欢得反复地听，甚至还设置成铃声。对我来说，遇到了这样的机会，我是以幸福的心情去参与的。

-最后请你向观众们进行结束问候。

在炎热的夏天里，真的哭了很多、也笑了很多幸福地进行了拍摄，通过《暂缓离别，一周》再一次感受到有多辛苦就有多珍贵的意义，如果能让大家产生共鸣的话，我会很感激，希望大家多多观看。谢谢！
