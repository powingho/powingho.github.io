采访翻译 | 徐贤2017年MBC电视剧《小偷家伙，小偷大人》终映采访

2017-11-23 18 min read # 电视剧 Drama # 徐贤 Seohyun # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

2007年，16岁的徐贤以少女时代成员的身份出道，之后通过发表《重逢的世界》《Gee》《Hoot》等热门歌曲在海内外受到欢迎。2013年开始，通过出演音乐剧和电视剧开始演技活动，而且最近在MBC周末剧《小偷大人小偷家伙》中饰演姜少珠第一次挑战地上波的女主角。虽然同时有着第一次担当主演的忧虑和期待，但徐贤自然的演技也得到了好评。尤其是和池贤宇搭档的爱情戏收获喜爱，得到了“姜石情侣”这样的爱称。电视剧结束后，与SM的专属合约也到期了，她独立了出来。

“过去的时间里，不仅仅是因为电视剧，进行着少女时代活动的这10年能够休息的时间并不是很多。这次绝对要好好地休息一下。国内要到处去转一转，去日本泡个温泉好好地放松一下。休息着应该可以发现自己新的模样。以前要是休息的话，会感到不安。因为不是只做一件事，拍摄电视剧、进行着少女时代的活动和音乐剧的活动，一直都是做着几件事。这次则是第一次可以集中在一件事上，少女时代10周年活动结束后，只集中在拍电视剧上，电视剧结束后可以集中在动着的人类徐珠贤上。明白了什么事都不干也不会感到不安，好像心里更从容了。一直都是看着眼前的东西活着，没有了那些东西后，才可以从容。”

“SM像家人一样已经15年了，出道前5年，少女时代活动了10年。各种感情都有，和像我的家人、娘家一样的SM不是离别地离别了。有各种各样的想法。决心离开的关键是不想安于现状，太好的环境，作为组合忙内得到的保护和喜爱，这些都很感谢。但另一方面是，觉得我已经不是10代了，处在20代的后半段，再过几年就要到30代了。回想了很多次自己的人生，也问了自己很多次以后想如何生活下去。放下所有的事情，我的人生我自己来负责，想这样生活看看，有了这样的想法。”

10年间一起的少女时代，对于忙内的独立有怎样的反应？

“我们之间真的是会聊很多的类型。最近也发了10周年的专辑，每天会聊各自的梦想，聊这样那样很多的事情。10年间一起成长过来，那些日子都一起度过，大家都非常了解彼此希望的人生方向，一直互相尊重着。因为这样，我们互相理解并且说了会为彼此应援到最后。决定离开SM的时候，也没有强迫或是反对这样的事。”

少女时代徐贤选择独立，Tiffany和秀英也没有延长合约。以后少女时代会怎样？

“不管怎么说，我现在希望更多地尝试演技，也会以歌手身份展示好的面貌，希望能够展示不同类型的模样。不是要设立个人所属社，现在为止还没有决定这些，各种可能性都会有，应该暂时会和Agency一起进行活动。而少女时代应该会是永远的，就像我们一直呼喊的口号‘现在是少女时代、以后是少女时代、永远是少女时代’，少女时代是永远的。对于活动方向，现在无法说明，之后活动的时候会再告诉大家。不管是什么形式，我们都会不变地为彼此应援。30岁也好，40岁也好，年岁增长了我们还是少女时代。到了那时，即便有人会说‘难道你们还是少女时代吗’，对我们来说，作为少女时代诞生了，将来也想这样（作为少女时代）帅气地变老。”

为什么离开SM？

“不久前和李秀满老师单独吃了饭，他感谢我到现在为止都做得很好，也给了我很多应援。把我从真的什么都不是的平凡少女培养成为少女时代这样有大众影响力的歌手，真的非常感谢他。不管去到哪里都要带着那份责任感活着。和很多人都聊过，但这样的决定完全是由我来做出的。到现在为止，得到了很多人的帮助，我只要做好自己的事就可以了。非常非常感谢，是获得福气的环境，但突然之间有了这里是不是太安逸了的想法，也稍微觉得不安。我真正想做的是什么，我将来想做的是什么，想着这些的时候，有了希望可以放下一切从头来过的想法。很难立刻得到答案，所以做了这样的选择。过了一段时间，一天一天地过着好像有了更多感受。产生了创造我自己主导的人生这样的想法，所以就做了这样的决定。”

SM对徐贤来说有着怎样的意义？从那里离开的感受如何？

“我的人生在进入SM之前和进入之后好像是分开的。过去10年在我人生当中真的是非常灿烂的瞬间，和少女时代一起度过了人生一半的时间。作为平凡人、作为歌手经历了很多，金钱都无法换来的经历，积累了许多养分的时间。独自开始的话，比起害怕，更多的感觉是巨大的责任感。以前是我们8个人分担的责任感，现在一下子全都落在了我的肩膀上，即便这样也是觉得很激动，紧张和激动共存的感觉。不管怎样，因为是我自己做的决定，所以没有后悔。更多地描绘未来的话，我对全新开始的人生感到激动。虽然不断苦恼着如何更帅气地活着，但没有得到正答就那样去尝试看看。再多学习一些的话，10年后成为比现在更帅气的女人就好了。希望没有后悔，带着对自己的责任感活着。”

不管如何，她的根都是少女时代，回想起少女时代过去10年的活动，她的眼神里透露着激动。

“和少女时代欧尼们一起长大，分享了所有的思春期，也有着所有敏感的时期。觉得能够彼此陪伴真的很万幸，给予彼此好的影响，也因此学到了很多。少女时代会继续的，会一直寻找能够一起活动的方法。现在也和欧尼们聊着出道15年的时候我们要一起做些什么。”

虽然离开SM，但和少女时代成员们现在仍在一起。

“15年一直在一起，大家都知道也不是必须保持联系，但我们还是会常常联系，团体聊天室一直都在。今天在美容室遇到孝渊欧尼也是聊个没完。不久前电视剧结束的时候，把最后一集的剧本拍下来传了上去，她们都说辛苦了。电视剧开始的时候，少女时代成员们也是一起准备，欧尼们很心疼，给我应援。很快50集结束了，大家都祝贺我。10年前和10年后的活动不会完全一样，所以虽然现在无法说明活动的方向性，但我们之间一直都有聊到这些。在韩国歌谣界，女团能走过10年这件事，我们也觉得自己很了不起，彼此夸赞。正因为如此，对于怎样维持下去比较好，不管是公司还是我们都一直在苦恼着。”

10年期间应该也包括了辛苦的时间。

“从某些角度来看，分开的时间也很好。5年以上的宿舍生活，越来越了解彼此，碰撞间逐渐成长。那些瞬间是最辛苦的时候，大家都有这样的共感。真的只和一个非常亲近的朋友一起生活都会有争执，何况我们有8个人。刚开始的时候真的很鸡毛蒜皮的事都能吵起来，现在回想起来是可以笑得出来，但在当时感受完全不同。大家的生活方式不同，真的有很多争吵。但也因为这样，变得更了解彼此。之后从宿舍生活转变成和各自和父母一起在附近生活了。虽然在一起生活的时候觉得很辛苦，但分开了反而更怀念住在一起的时候，很难过。这之后隔了一段时间再见面就觉得非常开心，好像变成了我们的日常。我们之间也聊到了很细小的未来，类似“几十年过后应该都结婚了吧”“你有了孩子之后会和你一起抚养”这些日常的对话。”

经历了所有的时间，10年里可以在一起的秘诀是什么？

“对话是很重要的。每天5分钟一定要对话是其中一条规则，这样下来不管再忙都能够稍微说上话，彼此对话的时间很重要。很多碰撞好像也是重要的，虽然可能会有争吵的时候，但不是看不到对方，而是要继续在一起，这样自然地会和好，感情也会变得更牢固。”

徐贤给人留下了既是少女时代忙内也是正直生活的少女的印象。

“非常感谢觉得我是很好的形象，但我所拥有的许多模样中，好像只喜欢其中一种而已，那并不是我的全部耶。。不过因为是非常大的肯定，有着那部分的存在，可以带着更多的责任感和感谢去工作。我还有着更多面的模样，希望将来可以一个一个地展示出来。”

徐贤在少女时代出道的时候只有16岁，对年纪还小就出道的她，有正直生活、少说话、文静的形象的理由是什么？

“说实话，我没有想过要因为正直生活的形象而要循规蹈矩地去生活，我玩的时候也玩，没有束缚自己。但在出道初期，确实比现在对自己更为严格一些。当时那样做的理由是，提前进入了社会生活，从父母陪着去学校，在规定的时间学习，去补习班这样的生活，变成了24小时都得靠着意志度过。所以刚开始的时候，每天都在跑行程，慢慢地昨天做了什么完全想不起来，太忙碌了感觉没有了自己的人生，觉得继续这样下去不行，所以从那时起就打起精神开始了（严格自我管理）。所有都是我的选择，10年、20年积累下来就会不一样。刚开始的时候，因为对自己感到不安，想要控制好自己，就更严格了。”

但是足足10年间以同一种形象度过会觉得郁闷吗？

“认真地（自我管理）过了几年，偶尔会感到很郁闷。看来我是对自己太松懈了吧，当初为了要控制自己才开始的，现在控制得太好了吧。现在觉得是可以相信自己的时候了，那时开始就有了想稍微松懈一下的想法，比起以前稍微放松了对自己的要求，所以也得到了很多教训。虽然很惧怕什么事都不做地打发时间，但感觉得有这样的时间才能接收新的事物，也可以refresh自己。现在想来也没有后悔。”

和看起来的形象不同，实际上会开很多玩笑，一直笑着，看起来更自由的她表情也变得明亮。

“平时也是会经常开玩笑，过去没有机会展现这方面的样子给大家。固定出演的综艺也是在20代初期的《我们结婚了》，7年的时间过去了，很多变化都没能展现出来，大家好像都还保留着《我结》那时的印象。不久前《认哥》播出的时候，好像吓到了大家，现在的我是那样的。真挚的时候很真挚，但基本上调皮的时候更多。要说《我结》那时年纪小很规律自己的话，而现在就觉得活着不必非得那样才行。更自由一些也可以。”

徐贤是少女时代当中唯一一个没有恋爱说的成员，“正直生活”的延长线多了“清净地域”这样的称呼。

“真的不太懂为什么那样。大家不都恋爱过吗，那样再正常不过了。作为一个女人是那样，作为一个平凡人也是那样，恋爱生活是理所当然的，不会不交往（与异性），20代后半了一次都没有恋爱过有人会相信吗？但我觉得人生的协调更重要，之前彻底贯彻自己的规则，随着时间过去，会有些许不同，虽然职业意识还是和以前一样，但人间烟火也是有必要的，不然的话不是太机器人了吗？以后大概我也会有恋爱说，虽然我希望尽最大努力不让这事发生。”

过去的夏天并行着电视剧的拍摄和少女时代的活动。为了拍摄MV做了华丽的美甲，回到拍摄现场在车里马上卸掉。熬夜也是家常便饭。

“我是少女时代的体力担当嘛。电视剧结束后，导演也认证了我的体力和精神力，但我觉得是得益于周围人们的帮助才能这样。”

徐贤在MBC周末剧《小偷》当中饰演姜少珠，完成了50集这么长的挑战。

“和优秀的人们共事是我的荣光。就像是50集这么长的合作，拍摄的时候觉得刚开始的负担感会消失，但事实不是那样，剧本通常是一周出两次，也有在拍摄前一天才出来的情况，6个月期间感觉每天都像是要考试一样，因此学习到了很多，有了更多的自信。在现场有很多变数，有天气的影响，也有剧本在现场修改很多的情况，因为罢工工作人员也换过，但是在那样的状况下最大限度地享受，所有人都努力了。到了真的要结束的时候就觉得时间过得很快，感觉像是拍了两三部mini电视剧，拍摄的时候也是不同的感受。遇到优秀的演员和工作人员们，愉快地进行了拍摄，学到的东西真的非常多。对我来说是非常有意义的作品。”

感谢对手戏的池贤宇。

“我觉得这次作品的对手戏演员碰得太好了，对池贤宇非常感谢。因为第一次主演很多负担，责任感也很重，和他对戏最多，所以得到了很多帮助，在现场让人觉得很方便地给予了帮助。我们年龄差很多，所以对同龄的设定有些担心，但他让我马上把话放下来（平语），我就被迫努力放下了。后来适应了，演技也变得自然了。真的是非常好的前辈。拍摄的时候经常向他请教，他用自身经历来回答。没有任何事是可以一个人可以完成的，所以感受到了责任感。”

两人的吻戏成为了话题。

“不是很过分，也不是很深的线。听说很多人喜欢，非常感谢大家。那样的吻戏是第一次，听到的是好的评价而不是觉得尴尬，觉得自己好像做得很好，很有意思。拍摄的时候，真的尽最大努力为了像恋爱一样投入进去而努力了。”

“真的很姜少珠”这样是很好的回馈吧，得益于打破端正善良的形象，融入充满正义感温暖的搜查官的角色中。

“收到这部作品的时候，听到很多说姜少珠这个角色并不适合我，是不是和我太不符合了，我觉得都是很正常的反应。对于大众来说，我的形象是正直生活的少女或是沉默文静的形象，要饰演这种强势角色应该会觉得很困惑吧。但我有信心，因为我性格当中有相当一部分就是那样的，觉得会变得有趣。”

对于徐贤来说，姜少珠像是活力素。成为展现自身拥有的另一种魅力的契机。

“做姜少珠的时候感觉是打开了喉咙。实际徐珠贤的性格中也有大大咧咧有趣的一面，那个部分通过姜少珠表现了出来。在扮演那个人物的时候，日常好像都变的粗糙起来，太有趣了。选择这部电视剧也是因为很满意角色设定，融为一体愉快地拍摄了。”

少女时代徐贤和演员徐珠贤的差别是什么？

“没有什么不同。少女时代徐贤、徐珠贤都是我，所以说并没有太大区别。以演员活动的时候用了徐珠贤这个名字，虽然是作为少女时代活动了10年的歌手，但演技这部分才刚刚开始，想带着更多初心去走，是这样的原因。下一步作品在商讨中，有机会的话也想挑战电影。虽然能收到下部作品是幸运也是福气，但觉得准备时间还是必要的。Solo专辑的可能性也有，会以歌手徐贤、演员徐贤向大家展现不同的模样。”

不管怎样，通过这次作品，得到了徐贤演技提升的好评。2013年在SBS《热爱》中饰演儿役第一次开始了演技，去年的《步步惊心》之后终于获得了演技好评。

“《步步惊心》算是第一部电视剧。不是儿役，第一次有了角色，为了不犯同样的错误，仔细地做了检查，努力地最大限度减少自己的缺点。非常感谢得到演技提高的评价。想要真诚地去演戏的想法通过这次作品做到了。现在还没有什么诀窍，那时就想着竭尽全力地努力去做。演得越多就能看到越多的不足之处，现在还差很远。虽然对年末演技大赏颁奖礼想说些什么，但没有对奖项的野心。能够演戏就很幸福。想带着强烈的责任感好好地完成。我的Role Model是金惠秀前辈和全度妍前辈，看了她们充满真诚地演技，我也想这么跟随着。”

今年对徐贤来说是特别的。

“2017年我真的做了很多事。发了第一张solo专辑，举行了第一次solo演唱会，出演了网剧，也发行了少女时代10周年的专辑，还有第一次主演了50集的电视剧，虽然没有停下来的时候，但比任何时候都有意义，每天都过得很有价值，因此也觉得很骄傲。开始了新的阶段和选择，应该会成为转折点。”

徐贤传达了Tiffany的近况。

“我很难说明Tiffany欧尼的计划，不管怎么说美国也是欧尼出生的地方，现在也在美国。以后可能在美国活动也可能回到韩国。”

接着说，“对于海外活动，我也很难说明。不管怎样，都有着报答得到的喜爱的想法。各自都会努力地活动。”

对于粉丝们，“不管怎么说都会觉得可惜，我也是那样。但是就像我的决心那样，会努力地展现好的面貌。还有因为少女时代会永远的，大家不要觉得太伤心就好了。将来少女时代的成员们不管怎样活动都会为彼此应援的。”

        • 徐贤是少女时代当中唯一一个没有恋爱说的成员，“正直生活”的延长线多了“清净地域”这样的称呼。
        • 过去的夏天并行着电视剧的拍摄和少女时代的活动。为了拍摄MV做了华丽的美甲，回到拍摄现场在车里马上卸掉。熬夜也是家常便饭。
        • 徐贤在MBC周末剧《小偷》当中饰演姜少珠，完成了50集这么长的挑战。
        • 感谢对手戏的池贤宇。
        • 两人的吻戏成为了话题。
        • 对于徐贤来说，姜少珠像是活力素。成为展现自身拥有的另一种魅力的契机。
        • 少女时代徐贤和演员徐珠贤的差别是什么？
        • 不管怎样，通过这次作品，得到了徐贤演技提升的好评。2013年在SBS《热爱》中饰演儿役第一次开始了演技，去年的《步步惊心》之后终于获得了演技好评。
        • 今年对徐贤来说是特别的。
        • 徐贤传达了Tiffany的近况。
