采访翻译｜崔秀英2021年杂志《Marie Claire》3月号

2021-02-23 7 min read # 奔向爱情 Run On # 杂志 Magazine # 秀英 Sooyoung # 少女时代 Girls' Generation SNSD # 采访 Interview # 翻译 Translation

Gender Free

希望更多的女演员们在作品中积累并存在各自的叙事，还有为了传达以期待展开更多女人的故事的心情和为演员们能够充分释放对演技的渴求、渴望应援的心情，和8位女演员一起的Gender Free 2021。

#崔秀英

#二十

#对我而言演员是自由

- 有没有看过以前Gender Free的企划呢？

去年看过之后我真的很想参与一次，所以今年收到提议觉得很神奇。在公演界，Gender Free进行选角并不活跃，所以以Gender Free完成选角的话剧《Orphans》让我印象很深刻，虽然剧中所有的出场人物都是男性，但女演员也一起被选入出演阵容了。我曾经想过男性人物由女人来表演是可能的吗？但公演界已经在实现多种多样的尝试，像这样通过杂志制作短视频也是很有意义的。

- 你表演了电影《二十》的台词，还有苦恼过的电影《阿修罗》里葬礼的场面。

《阿修罗》是对女演员来说机会并不多的黑色犯罪题材。坏人出场后，提出了“谁才是真正的坏人，什么才是真正的善”的问题，并引导了电影的发展，其中来吊丧的朴星培关于辣牛肉汤一番话虽然是非常常见的，但因为那是展现电影整体感觉的场面，所以我想试试看。越反复咀嚼，越觉得毛骨悚然，也有卑劣的感觉。那样卑劣、毛骨悚然地展现恶的极致的角色，我能够出演的机会不多，所以很好奇我演的话是什么样的感觉。相反，《二十》则是没有太严肃、比较轻松的，感觉会很有趣。

- 过去一年里让人印象深刻的女性角色尤其很多，其中《Run On》里的“丹雅”有种颠覆了男性财阀角色所带来的俗套的感觉。

饰演丹雅的时候，真的非常喜欢、也很刺激。比起别的，更让我喜欢的是她是一个努力工作的人物，不过我没有期待她只是符合这个时代女性形象的框架而有局限性的人物，我们所生活的这个世上真的有多种多样的女性存在，在那之中也有进取心强、不怎么了解主体价值观的人，那样的人也能成为主人公。如果能有展现那样主人公成长模样的作品，感觉会是很好的事。为了符合时代需求的女性形象而让故事线被破坏，我反而觉得是危险的。希望可以在那样的限制下创作更多丰富的人物。为了看到还没有成长、有着苦痛的角色因为什么契机而发出自己的声音，希望能出现更多没有成长的人物。我也想看到弱女子的成长故事。

- 最近看得津津有味的女人是谁呢？

是《后翼弃兵》，虽然在那个时代国际象棋是只属于男人的比赛，“贝斯·哈蒙”却赢了男性对手。但她也有生活的漏洞，沉迷药物抓住了她的精神、被家人抛弃也是重要的故事线、经历低谷后度过了精神不健康的时间。我不认为《后翼弃兵》是为了女性在国际象棋比赛中战胜男性带来痛快感的电视剧，而是觉得这是一部关于天才拥有天赋的同时也要付出代价的电视剧。只是，那个天才可以是男人，也可以是女人。

- 对这样的故事感兴趣的理由是什么呢？

我喜欢关于很有野心、也有能力的人的电视剧。有时是双方的拉锯战，有时是和权力斗争，然后向着目标前进，我会被这样的故事吸引。我希望在现实中也能成为抛开性别，仅仅因为这个人做得好而获得认可的社会。我希望这个世上能够变成我们所有人作为一个人而存在，没有歧视和逆向歧视，只以实力正正当当地获得评价。

- 采访前，拍摄定义演员的评论时，你定义了演员是自由。

表演似乎就是自由。表演的时候可以成为任何一个人，作为角色不管说什么话都可以得到原谅，还可以表达想获得自由的愿望。我希望可以在别人的视线和偏见中自由自在地表演、从在现场感到紧张的我身上自由自在地表演。从不同的意义上想到了自由这个单词。

- 即使是在《Run On》播出前和Marie Claire拍摄时候，作为演员好像有很多心动的感觉。相反的，与对于电视剧和丹雅热烈的反应对比，今天看起来特别的沉稳。

是的，那时候真的活在非常心动的时间里。结束电视剧后，现在的我没有那么激动了，现在好像不会再为了大事小事而悲喜交加，之前我曾有过的担忧和不安都消失了。以前我担心自己太过激动或太过冷静，而过分地督促自己，对于人们对我的评价心情也是在天堂和地狱之间来回，但现在更把这些工作当作职业来看待了。现在我觉得自己是可以做好才做的这些工作，这样想的话才不会太折磨自己。在一个作品里，成为团队的一员、担当作品的一部分是我的工作，那样像职业一样接受的健康想法才是工作的法则。

- 度过被填得满满年末迎来新年，你想怎么填满2021年呢？

我仍然是有很多希望经历的事。对国外的制作体系感到好奇，也对置于陌生环境里的我感到好奇。我是希望可以无穷无尽学习的人。作品结束后，这样休息的时间是有点辛苦的时期，有时也觉得茫然。现在我觉得没有被人选择的我也是我，如果说以前的我是强行用自尊心和自尊感来包装自己的不足，那么现在就是模样已经雕琢好的感觉。我可以带着由内心的确信带来的自尊感去工作了，没有被选择的时候也不会觉得是自己的不足，而是现在没有适合我的罢了。今年也会这样度过的。“丹雅”之所以珍贵的原因之一是让我感受到了满足，别人想从我这里看到的东西，首先想起自己不足的时候已经过去了，现在不会像被追赶一样做出急躁的选择。即使没有被选中，我也会发出自己的声音直到被人认出为止，这是我今年的计划。
